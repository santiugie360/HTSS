Imágenes en HTML

Aprende a incorporar y optimizar imágenes en tus páginas web

Introducción a las Imágenes

Las imágenes son elementos fundamentales en el diseño web moderno. Permiten transmitir información de forma visual, mejorar la experiencia del usuario y hacer que las páginas sean más atractivas y memorables.

Impacto Visual

Las imágenes captan la atención del usuario inmediatamente y comunican ideas de forma más rápida y efectiva que el texto.

Optimización

Las imágenes bien optimizadas mejoran el rendimiento de la página, reducen los tiempos de carga y ahorran ancho de banda.

Responsividad

Las imágenes responsivas se adaptan a diferentes tamaños de pantalla, garantizando una experiencia óptima en todos los dispositivos.

La Etiqueta <img>

Sintaxis básica

<img src="ruta-de-la-imagen.jpg" alt="Descripción de la imagen">

La etiqueta <img> es un elemento vacío (no tiene etiqueta de cierre) que inserta una imagen en la página web.

Características principales:

  • Es un elemento inline por defecto
  • No requiere etiqueta de cierre
  • Necesita al menos el atributo src
  • Por accesibilidad, debe incluir el atributo alt

Formatos de imagen comunes

JPEG/JPG

.jpg
  • Ideal para fotografías
  • Compresión con pérdida
  • No soporta transparencia
  • Millones de colores

PNG

.png
  • Ideal para gráficos
  • Compresión sin pérdida
  • Soporta transparencia
  • Archivos más pesados

GIF

.gif
  • Soporta animaciones
  • Máximo 256 colores
  • Transparencia limitada
  • Ideal para iconos simples

WebP

.webp
  • Formato moderno
  • Mejor compresión
  • Soporta transparencia
  • Soporta animaciones

Atributos de la Etiqueta <img>

Atributos esenciales

1 src

Especifica la ruta (URL) de la imagen. Es el único atributo obligatorio.

<img src="images/foto.jpg">

Puede ser una ruta relativa, absoluta o una URL completa.

2 alt

Proporciona un texto alternativo que se muestra si la imagen no puede cargarse. Crucial para accesibilidad.

<img src="logo.png" alt="Logo de la empresa">

Los lectores de pantalla utilizan este texto para describir la imagen a usuarios con discapacidad visual.

Atributos de dimensión

3 width y height

Especifican el ancho y alto de la imagen en píxeles o porcentaje.

<img src="banner.jpg" alt="Banner" width="800" height="400">

Definir estas dimensiones ayuda a prevenir el desplazamiento del contenido durante la carga de la página.

4 loading

Controla cómo el navegador debe cargar la imagen.

<img src="imagen.jpg" alt="Descripción" loading="lazy">

El valor "lazy" retrasa la carga de imágenes fuera de la pantalla hasta que el usuario se desplaza cerca de ellas.

Atributos adicionales

Atributo Descripción Ejemplo
title Muestra información adicional al pasar el cursor sobre la imagen. title="Fotografía del equipo"
decoding Indica cómo debe decodificarse la imagen (async, sync, auto). decoding="async"
crossorigin Especifica si la imagen debe cargarse con una solicitud CORS. crossorigin="anonymous"
ismap / usemap Utilizados para crear mapas de imagen (áreas clicables). usemap="#mapname"

Imágenes Responsivas

Las imágenes responsivas se adaptan a diferentes tamaños de pantalla y resoluciones, optimizando la experiencia del usuario y el rendimiento de la página.

Técnica 1: CSS max-width

La forma más sencilla de hacer que una imagen sea responsiva es utilizando CSS para limitar su ancho máximo.

HTML

<img src="imagen.jpg" alt="Descripción" class="responsive-img">

CSS

.responsive-img {
  max-width: 100%;
  height: auto;
}

Esta técnica hace que la imagen se reduzca para ajustarse a su contenedor, pero nunca se expandirá más allá de su tamaño original.

Técnica 2: Atributo srcset

El atributo srcset permite especificar diferentes versiones de una imagen para diferentes resoluciones de pantalla.

<img src="imagen-small.jpg"
     srcset="imagen-small.jpg 500w,
             imagen-medium.jpg 1000w,
             imagen-large.jpg 1500w"
     alt="Descripción">

El navegador elegirá automáticamente la imagen más adecuada según el tamaño de la ventana y la densidad de píxeles del dispositivo.

  • El valor w indica el ancho de cada imagen en píxeles
  • El atributo src funciona como respaldo para navegadores que no soportan srcset

Técnica 3: Elemento picture

El elemento <picture> ofrece un control más preciso sobre qué imagen mostrar en diferentes situaciones.

<picture>
  <source media="(max-width: 600px)" srcset="imagen-mobile.jpg">
  <source media="(max-width: 1200px)" srcset="imagen-tablet.jpg">
  <img src="imagen-desktop.jpg" alt="Descripción">
</picture>

El elemento <picture> permite:

  • Mostrar diferentes imágenes según el tamaño de la pantalla usando media
  • Ofrecer diferentes formatos de imagen con <source type="image/webp">
  • Siempre incluir un elemento <img> como fallback

Mejores prácticas para imágenes responsivas

Optimización

  • Comprime las imágenes sin perder calidad visible
  • Utiliza formatos modernos como WebP cuando sea posible
  • Considera usar servicios de CDN para imágenes
  • Elimina metadatos innecesarios de las imágenes

Rendimiento

  • Usa loading="lazy" para imágenes bajo el pliegue
  • Proporciona dimensiones width y height para evitar saltos de diseño
  • Considera usar miniaturas o imágenes de baja resolución como placeholders
  • No escales imágenes más grandes de lo necesario
