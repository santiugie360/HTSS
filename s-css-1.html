Introducción a CSS

Cascading Style Sheets (CSS) es el lenguaje que da estilo y diseño a tus páginas web, transformando el contenido HTML en experiencias visuales atractivas.

Características de CSS

Lenguaje de Estilos

CSS no es un lenguaje de programación; es un lenguaje de hojas de estilo que define la apariencia de tu contenido HTML. CSS consiste en una serie de reglas que determinan cómo se mostrarán los elementos en la pantalla.

Con CSS puedes controlar colores, fuentes, márgenes, posicionamiento, animaciones y mucho más, permitiéndote crear diseños atractivos y responsivos.

Separación de Contenido y Diseño

CSS permite separar el contenido (HTML) de su presentación visual. Esta separación hace que el mantenimiento de sitios web sea más sencillo, ya que puedes modificar el diseño sin alterar el contenido.

Un solo archivo CSS puede controlar el estilo de múltiples páginas HTML, lo que garantiza consistencia visual y reduce la duplicación de código.

Selectores y Especificidad

CSS utiliza selectores para apuntar a elementos HTML específicos. Puedes seleccionar elementos por tipo, clase, ID, atributo o relación con otros elementos.

La especificidad determina qué reglas CSS se aplican cuando múltiples reglas apuntan al mismo elemento, creando un sistema de cascada que da nombre al lenguaje.

Diseño Responsivo

CSS moderno permite crear diseños que se adaptan a diferentes tamaños de pantalla y dispositivos mediante técnicas como media queries, flexbox y grid.

Estas herramientas permiten que un sitio web se vea bien y funcione correctamente en dispositivos que van desde teléfonos móviles hasta pantallas de escritorio de gran tamaño.

Historia y Evolución

Desde su creación hasta la actualidad, CSS ha evolucionado significativamente

1996: CSS Nivel 1

Primera especificación oficial de CSS. Introdujo propiedades básicas para fuentes, colores, texto, márgenes, bordes y posicionamiento.

1
2

1998: CSS Nivel 2

Añadió soporte para medios específicos (impresión, pantalla), posicionamiento absoluto, relativo y fijo, y el concepto de z-index.

2011: CSS 2.1

Corrección y actualización de CSS2. Se convirtió en la especificación estable y ampliamente implementada por los navegadores.

3
4

2012-presente: CSS3 y Módulos

CSS3 se dividió en módulos independientes. Introdujo animaciones, transiciones, flexbox, grid, variables, media queries y muchas más características avanzadas.

Ejemplos de Código CSS

CSS Básico

/* Selector de elemento */
body {
  font-family: 'Arial', sans-serif;
  background-color: #f0f0f0;
  margin: 0;
  padding: 0;
}

/* Selector de clase */
.container {
  max-width: 1200px;
  margin: 0 auto;
  padding: 20px;
}

/* Selector de ID */
#header {
  background-color: #8b5cf6;
  color: white;
  padding: 1rem;
}

CSS Avanzado

/* Flexbox */
.flex-container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  flex-wrap: wrap;
}

/* Media Query */
@media (max-width: 768px) {
  .flex-container {
    flex-direction: column;
  }
}

/* Animación */
@keyframes fadeIn {
  0% { opacity: 0; }
  100% { opacity: 1; }
}

.animated {
  animation: fadeIn 1s ease-in-out;
}
