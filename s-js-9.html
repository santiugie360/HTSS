JavaScript Events

Los eventos son acciones o sucesos que ocurren en el navegador, como clics, movimientos del mouse o pulsaciones de teclas. JavaScript puede "escuchar" estos eventos y ejecutar código en respuesta, creando experiencias web interactivas y dinámicas.

¿Qué son los Eventos?

Interacción del Usuario

Los eventos permiten capturar las acciones del usuario como clics en botones, escritura en formularios, movimientos del mouse y mucho más. Sin eventos, las páginas web serían estáticas.

Cada vez que un usuario interactúa con tu página, se dispara un evento que JavaScript puede detectar y manejar con addEventListener().

Event Listeners

Un event listener (oyente de eventos) es una función que "escucha" un evento específico en un elemento. Cuando el evento ocurre, se ejecuta el código dentro del listener.

Puedes agregar múltiples listeners al mismo elemento para diferentes eventos, proporcionando gran flexibilidad y control sobre el comportamiento de tu aplicación.

Event Object

Cuando ocurre un evento, JavaScript crea automáticamente un objeto de evento que contiene información útil: qué elemento disparó el evento, coordenadas del mouse, tecla presionada, etc.

Este objeto se pasa como parámetro a la función listener y puedes acceder a propiedades como event.target, event.preventDefault() y más.

Event Propagation

Los eventos se propagan a través del DOM en dos fases: bubbling (de hijo a padre) y capturing (de padre a hijo). Esto permite capturar eventos en elementos padres para múltiples hijos.

Puedes controlar esta propagación con métodos como stopPropagation() para evitar que el evento suba por el árbol del DOM.

Tipos de Eventos

Conoce los tipos de eventos más comunes en JavaScript

Eventos del Mouse

click

element.addEventListener('click', (e) => {
  console.log('¡Clic!');
});

Se dispara cuando el usuario hace clic en un elemento.

dblclick

element.addEventListener('dblclick', (e) => {
  console.log('¡Doble clic!');
});

Se dispara cuando el usuario hace doble clic.

mouseenter / mouseleave

element.addEventListener('mouseenter', () => {
  console.log('Mouse dentro');
});
element.addEventListener('mouseleave', () => {
  console.log('Mouse fuera');
});

Se disparan cuando el mouse entra o sale de un elemento.

mousemove

element.addEventListener('mousemove', (e) => {
  console.log(`X: ${e.clientX}, Y: ${e.clientY}`);
});

Se dispara cuando el mouse se mueve sobre un elemento.

contextmenu

element.addEventListener('contextmenu', (e) => {
  e.preventDefault();
  console.log('Clic derecho');
});

Se dispara cuando se hace clic derecho (menú contextual).

Eventos del Teclado

keydown

document.addEventListener('keydown', (e) => {
  console.log(`Tecla: ${e.key}`);
});

Se dispara cuando una tecla es presionada.

keyup

document.addEventListener('keyup', (e) => {
  console.log('Tecla liberada');
});

Se dispara cuando una tecla es liberada.

keypress (deprecated)

// Ya no se recomienda usar keypress
// Usar keydown en su lugar

Evento obsoleto, usar keydown en su lugar.

Propiedades útiles

document.addEventListener('keydown', (e) => {
  console.log(e.key);      // 'a', 'Enter', etc.
  console.log(e.code);     // 'KeyA', 'Enter', etc.
  console.log(e.ctrlKey);  // true si Ctrl está presionado
  console.log(e.shiftKey); // true si Shift está presionado
});

Propiedades del evento de teclado.

Eventos de Formularios

submit

form.addEventListener('submit', (e) => {
  e.preventDefault();
  console.log('Formulario enviado');
});

Se dispara cuando se envía un formulario.

input

input.addEventListener('input', (e) => {
  console.log(e.target.value);
});

Se dispara cada vez que cambia el valor de un input.

change

select.addEventListener('change', (e) => {
  console.log(e.target.value);
});

Se dispara cuando un elemento pierde el foco después de cambiar.

focus / blur

input.addEventListener('focus', () => {
  console.log('Input enfocado');
});
input.addEventListener('blur', () => {
  console.log('Input desenfocado');
});

Se disparan cuando un elemento gana o pierde el foco.

Eventos de Window/Document

DOMContentLoaded

document.addEventListener('DOMContentLoaded', () => {
  console.log('DOM cargado');
});

Se dispara cuando el HTML está completamente cargado y parseado.

load

window.addEventListener('load', () => {
  console.log('Página completamente cargada');
});

Se dispara cuando toda la página (incluyendo recursos) está cargada.

resize

window.addEventListener('resize', () => {
  console.log(`Ancho: ${window.innerWidth}`);
});

Se dispara cuando se redimensiona la ventana del navegador.

scroll

window.addEventListener('scroll', () => {
  console.log(window.scrollY);
});

Se dispara cuando el usuario hace scroll en la página.

Ejemplos Prácticos

Contador de Clics

// HTML: <button id="btn">Clic aquí</button> <span id="contador">0</span>

let count = 0;
const button = document.getElementById('btn');
const contador = document.getElementById('contador');

button.addEventListener('click', () => {
  count++;
  contador.textContent = count;
});

Ejemplo básico de evento click que incrementa un contador.

Validación en Tiempo Real

// HTML: <input id="email" type="email"> <span id="error"></span>

const email = document.getElementById('email');
const error = document.getElementById('error');

email.addEventListener('input', (e) => {
  const value = e.target.value;
  const regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  
  if (!regex.test(value) && value !== '') {
    error.textContent = 'Email inválido';
  } else {
    error.textContent = '';
  }
});

Validar un email en tiempo real mientras el usuario escribe.

Hover Effect

// HTML: <div id="box" style="width:100px;height:100px;background:blue"></div>

const box = document.getElementById('box');

box.addEventListener('mouseenter', () => {
  box.style.backgroundColor = 'red';
  box.style.transform = 'scale(1.2)';
});

box.addEventListener('mouseleave', () => {
  box.style.backgroundColor = 'blue';
  box.style.transform = 'scale(1)';
});

Cambiar estilos cuando el mouse entra y sale de un elemento.

Prevenir Submit de Formulario

// HTML: <form id="form"><input type="text" required><button>Enviar</button></form>

const form = document.getElementById('form');

form.addEventListener('submit', (e) => {
  e.preventDefault(); // Evita recargar la página
  
  const formData = new FormData(form);
  console.log('Datos del formulario:', formData);
  
  // Aquí puedes enviar los datos con fetch
});

Prevenir el comportamiento por defecto y manejar el submit con JavaScript.

Event Delegation (Delegación de Eventos)

// HTML: <ul id="lista"><li>Item 1</li><li>Item 2</li><li>Item 3</li></ul>

const lista = document.getElementById('lista');

// En lugar de agregar un listener a cada <li>, lo agregamos al padre
lista.addEventListener('click', (e) => {
  // Verificamos si el elemento clickeado es un <li>
  if (e.target.tagName === 'LI') {
    console.log('Clickeaste:', e.target.textContent);
    e.target.style.backgroundColor = 'yellow';
  }
});

// Agregar nuevos elementos dinámicamente también funcionará
const newItem = document.createElement('li');
newItem.textContent = 'Item nuevo';
lista.appendChild(newItem);

La delegación de eventos es una técnica eficiente para manejar eventos en múltiples elementos similares, especialmente cuando se agregan dinámicamente.
