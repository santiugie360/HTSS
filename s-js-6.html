JavaScript Arrays

Los arrays son estructuras de datos fundamentales que permiten almacenar múltiples valores en una única variable. Son ordenados, dinámicos y ofrecen una amplia variedad de métodos para manipular datos de manera eficiente y elegante.

¿Qué son los Arrays?

Colección Ordenada

Un array es una lista ordenada de elementos donde cada elemento tiene una posición numérica llamada índice. El primer elemento está en el índice 0, el segundo en el 1, y así sucesivamente.

Ejemplo: ['manzana', 'banana', 'naranja'] tiene 3 elementos en las posiciones 0, 1 y 2.

Tipos Mixtos

A diferencia de otros lenguajes, los arrays de JavaScript pueden contener diferentes tipos de datos en la misma estructura: números, strings, objetos, otros arrays, etc.

Ejemplo: [42, 'texto', true, {nombre: 'Ana'}] es perfectamente válido.

Tamaño Dinámico

Los arrays en JavaScript crecen y se reducen automáticamente. No necesitas declarar su tamaño inicial y puedes agregar o eliminar elementos en cualquier momento.

Puedes agregar elementos con push() o eliminarlos con pop(), y el tamaño se ajusta automáticamente.

Métodos Poderosos

JavaScript proporciona más de 30 métodos integrados para manipular arrays: filtrar, mapear, reducir, ordenar, buscar, y mucho más. Estos métodos hacen que trabajar con arrays sea muy eficiente.

Métodos modernos como map(), filter() y reduce() permiten transformar datos de manera elegante.

Acceder a Elementos:

const frutas = ['manzana', 'banana', 'naranja'];

// Acceso por índice
console.log(frutas[0]); // 'manzana'

Métodos Básicos

Métodos fundamentales para manipular arrays

push()

Agrega elementos al final del array

const arr = [1, 2];
arr.push(3);
console.log(arr); 
// [1, 2, 3]

pop()

Elimina el último elemento

const arr = [1, 2, 3];
const ultimo = arr.pop();
console.log(ultimo); // 3
console.log(arr); // [1, 2]

unshift()

Agrega elementos al inicio

const arr = [2, 3];
arr.unshift(1);
console.log(arr); 
// [1, 2, 3]

shift()

Elimina el primer elemento

const arr = [1, 2, 3];
const primero = arr.shift();
console.log(primero); // 1
console.log(arr); // [2, 3]

slice()

Copia una porción del array

const arr = [1, 2, 3, 4];
const porcion = arr.slice(1, 3);
console.log(porcion); 
// [2, 3]

splice()

Agrega/elimina elementos

const arr = [1, 2, 3];
arr.splice(1, 1, 99);
console.log(arr); 
// [1, 99, 3]

concat()

Une dos o más arrays

const arr1 = [1, 2];
const arr2 = [3, 4];
const union = arr1.concat(arr2);
console.log(union); 
// [1, 2, 3, 4]

join()

Une elementos en un string

const arr = ['Hola', 'Mundo'];
const str = arr.join(' ');
console.log(str); 
// 'Hola Mundo'

indexOf()

Busca el índice de un elemento

const arr = [1, 2, 3];
const idx = arr.indexOf(2);
console.log(idx); 
// 1

includes()

Verifica si existe un elemento

const arr = [1, 2, 3];
console.log(arr.includes(2)); 
// true

reverse()

Invierte el orden del array

const arr = [1, 2, 3];
arr.reverse();
console.log(arr); 
// [3, 2, 1]

sort()

Ordena los elementos

const arr = [3, 1, 2];
arr.sort();
console.log(arr); 
// [1, 2, 3]

Arrow Functions (=>)

Funciones flecha: sintaxis moderna y concisa para escribir funciones

¿Qué son las Arrow Functions?

Las arrow functions son una sintaxis más corta para escribir funciones introducida en ES6. Son especialmente útiles cuando se trabaja con métodos de arrays como map(), filter(), y reduce().

Ventajas:

  • •
    Sintaxis más corta y legible
  • •
    Return implícito en una línea
  • •
    No tienen su propio this
  • •
    Perfectas para callbacks

Comparación de Sintaxis:

// Función tradicional
function sumar(a, b) {
  return a + b;
}

// Arrow function equivalente
const sumar = (a, b) => a + b;

// Con un solo parámetro (sin paréntesis)
const doble = n => n * 2;

// Sin parámetros
const saludar = () => 'Hola';

// Con cuerpo de función (necesita return)
const calcular = (x, y) => {
  const suma = x + y;
  return suma * 2;
};

Un Solo Parámetro

// Sin paréntesis
const cuadrado = n => n * n;

// Con paréntesis (opcional pero válido)
const cubo = (n) => n * n * n;

console.log(cuadrado(5)); // 25
console.log(cubo(3)); // 27

Múltiples Parámetros

// Paréntesis obligatorios
const multiplicar = (a, b) => a * b;

const concatenar = (str1, str2, separador) => 
  str1 + separador + str2;

console.log(multiplicar(4, 5)); // 20
console.log(concatenar('Hola', 'Mundo', ' ')); 
// 'Hola Mundo'

Return Implícito

// Return implícito (una línea)
const obtenerNombre = persona => persona.nombre;

// Return de objeto (requiere paréntesis)
const crearPersona = (nombre, edad) => ({
  nombre: nombre,
  edad: edad
});

console.log(crearPersona('Ana', 25));
// { nombre: 'Ana', edad: 25 }

Return Explícito

// Múltiples líneas necesitan llaves y return
const calcularDescuento = (precio, porcentaje) => {
  const descuento = precio * (porcentaje / 100);
  const precioFinal = precio - descuento;
  return precioFinal;
};

console.log(calcularDescuento(100, 20)); 
// 80

Métodos Avanzados

Métodos poderosos que transforman y procesan arrays con arrow functions

🗺️

map()

Transforma cada elemento del array y devuelve un nuevo array

Descripción:

map() crea un nuevo array aplicando una función a cada elemento del array original. No modifica el array original.

Cuándo usar: Cuando necesites transformar todos los elementos de un array de la misma manera (duplicar números, extraer propiedades, formatear datos). 

// Duplicar cada número
const numeros = [1, 2, 3, 4];
const dobles = numeros.map(n => n * 2);
console.log(dobles); // [2, 4, 6, 8]

// Extraer nombres de objetos
const personas = [
  { nombre: 'Ana', edad: 25 },
  { nombre: 'Juan', edad: 30 }
];
const nombres = personas.map(p => p.nombre);
console.log(nombres); // ['Ana', 'Juan']

// Formatear precios
const precios = [10, 25.5, 99.99];
const formateados = precios.map(p => '$' + p.toFixed(2));
console.log(formateados); 
// ['$10.00', '$25.50', '$99.99']
🔍

filter()

Filtra elementos que cumplan una condición

Descripción:

filter() crea un nuevo array con todos los elementos que pasen una prueba (función que devuelve true/false).

Cuándo usar: Cuando necesites seleccionar solo algunos elementos que cumplan ciertas condiciones (filtrar por edad, buscar elementos específicos, eliminar valores nulos). 

// Números pares
const numeros = [1, 2, 3, 4, 5, 6];
const pares = numeros.filter(n => n % 2 === 0);
console.log(pares); // [2, 4, 6]

// Personas mayores de edad
const personas = [
  { nombre: 'Ana', edad: 17 },
  { nombre: 'Juan', edad: 25 },
  { nombre: 'María', edad: 16 }
];
const mayores = personas.filter(p => p.edad >= 18);
console.log(mayores); 
// [{ nombre: 'Juan', edad: 25 }]

// Filtrar strings largos
const palabras = ['hola', 'mundo', 'JavaScript'];
const largas = palabras.filter(p => p.length > 5);
console.log(largas); // ['JavaScript']
📊

reduce()

Reduce el array a un único valor

Descripción:

reduce() ejecuta una función reductora sobre cada elemento, devolviendo un único valor resultante. Es el método más versátil.

Cuándo usar: Cuando necesites combinar todos los elementos en un solo valor (sumar, contar, agrupar, construir objetos). 

// Sumar todos los números
const numeros = [1, 2, 3, 4];
const suma = numeros.reduce((acc, n) => acc + n, 0);
console.log(suma); // 10

// Encontrar el máximo
const max = numeros.reduce((acc, n) => 
  n > acc ? n : acc
);
console.log(max); // 4

// Contar ocurrencias
const frutas = ['🍎', '🍌', '🍎', '🍊', '🍌', '🍎'];
const conteo = frutas.reduce((acc, fruta) => {
  acc[fruta] = (acc[fruta] || 0) + 1;
  return  + 1;
  return acc;
}, {});
console.log(conteo); 
// { 🍎: 3, 🍌: 2, 🍊: 1 }
🎯

find() / findIndex()

Encuentra el primer elemento que cumple una condición

Descripción:

find() devuelve el primer elemento que cumple la condición. findIndex() devuelve su índice.

Cuándo usar: Cuando necesites encontrar un elemento específico en un array (buscar un usuario por ID, encontrar el primer elemento que cumpla una condición). 

// Encontrar primer número mayor a 10
const numeros = [5, 8, 12, 15, 20];
const encontrado = numeros.find(n => n > 10);
console.log(encontrado); // 12

// Encontrar índice
const indice = numeros.findIndex(n => n > 10);
console.log(indice); // 2

// Buscar objeto por propiedad
const usuarios = [
  { id: 1, nombre: 'Ana' },
  { id: 2, nombre: 'Juan' }
];
const usuario = usuarios.find(u => u.id === 2);
console.log(usuario.nombre); // 'Juan'
✅

some() / every()

Verifica si alguno o todos los elementos cumplen una condición

Descripción:

some() devuelve true si AL MENOS UN elemento cumple la condición. every() devuelve true si TODOS los elementos la cumplen.

Cuándo usar: Para validaciones (verificar si hay elementos que cumplan una condición, validar que todos los elementos sean válidos). 

const edades = [15, 18, 22, 30];

// ¿Hay algún mayor de edad?
const hayMayores = edades.some(edad => edad >= 18);
console.log(hayMayores); // true

// ¿Todos son mayores de edad?
const todosMayores = edades.every(edad => edad >= 18);
console.log(todosMayores); // false

// Validar array de números positivos
const numeros = [5, 10, 15];
const todosPositivos = numeros.every(n => n > 0);
console.log(todosPositivos); // true
🔄

forEach()

Ejecuta una función para cada elemento

Descripción:

forEach() ejecuta una función para cada elemento del array. No devuelve nada (undefined).

Cuándo usar: Para efectos secundarios (mostrar datos, hacer peticiones, modificar el DOM). Si necesitas un nuevo array, usa map(). 

// Imprimir cada elemento
const frutas = ['🍎', '🍌', '🍊'];
frutas.forEach(fruta => {
  console.log('Me gusta ' + fruta);
});

// Con índice
frutas.forEach((fruta, indice) => {
  console.log(`${indice + 1}. ${fruta}`);
});

// Modificar DOM (ejemplo)
const numeros = [1, 2, 3];
numeros.forEach(num => {
  // document.body.innerHTML += `<p>${num}</p>`;
});

Ejemplos Prácticos

Cadena de Métodos (Method Chaining)

// Filtrar, transformar y sumar
const productos = [
  { nombre: 'Laptop', precio: 1000, stock: 5 },
  { nombre: 'Mouse', precio: 25, stock: 0 },
  { nombre: 'Teclado', precio: 75, stock: 10 }
];

const total = productos
  .filter(p => p.stock > 0) // Solo disponibles
  .map(p => p.precio) // Solo precios
  .reduce((sum, precio) => sum + precio, 0); // Sumar

console.log(total); // 1075

Combinar múltiples métodos para procesar datos de manera eficiente.

Agrupar Datos

// Agrupar personas por edad
const personas = [
  { nombre: 'Ana', edad: 25 },
  { nombre: 'Juan', edad: 25 },
  { nombre: 'María', edad: 30 }
];

const porEdad = personas.reduce((grupos, persona) => {
  const edad = persona.edad;
  if (!grupos[edad]) {
    grupos[edad] = [];
  }
  grupos[edad].push(persona);
  return grupos;
}, {});

console.log(porEdad);
// { 25: [{Ana}, {Juan}], 30: [{María}] }

Usar reduce para agrupar elementos por una propiedad.

Eliminar Duplicados

// Método 1: Con Set
const numeros = [1, 2, 2, 3, 3, 4];
const unicos = [...new Set(numeros)];
console.log(unicos); // [1, 2, 3, 4]

// Método 2: Con filter
const unicos2 = numeros.filter((num, index) => 
  numeros.indexOf(num) === index

// Eliminar duplicados de objetos
const personas = [
  { id: 1, nombre: 'Ana' },
  { id: 2, nombre: 'Juan' },
  { id: 1, nombre: 'Ana' }
];
const unicasPersonas = personas.filter((p, i, arr) =>
  arr.findIndex(t => t.id === p.id) === i
);

Diferentes técnicas para eliminar elementos duplicados.

Ordenamiento Avanzado

// Ordenar números (sort por defecto es alfabético)
const numeros = [10, 5, 40, 25];
numeros.sort((a, b) => a - b); // Ascendente
console.log(numeros); // [5, 10, 25, 40]

// Descendente
numeros.sort((a, b) => b - a);

// Ordenar objetos por propiedad
const productos = [
  { nombre: 'Laptop', precio: 1000 },
  { nombre: 'Mouse', precio: 25 },
  { nombre: 'Teclado', precio: 75 }
];

// Por precio
productos.sort((a, b) => a.precio - b.precio);

// Por nombre (alfabético)
productos.sort((a, b) => 
  a.nombre.localeCompare(b.nombre)
);

Ordenar arrays de números y objetos correctamente.

Transformación Completa de Datos

// Datos de entrada
const pedidos = [
  { id: 1, cliente: 'Ana', items: [100, 200], estado: 'completado' },
  { id: 2, cliente: 'Juan', items: [50], estado: 'pendiente' },
  { id: 3, cliente: 'María', items: [300, 150, 75], estado: 'completado' }
];

// Calcular total de ventas completadas
const totalVentas = pedidos
  .filter(p => p.estado === 'completado')
  .map(p => p.items.reduce((sum, item) => sum + item, 0))
  .reduce((total, pedidoTotal) => total + pedidoTotal, 0);

console.log(totalVentas); // 825

// Crear reporte por cliente
const reporte = pedidos.map(pedido => ({
  cliente: pedido.cliente,
  total: pedido.items.reduce((sum, item) => sum + item, 0),
  cantidadItems: pedido.items.length,
  estado: pedido.estado
}));

console.log(reporte);
// [
//   { cliente: 'Ana', total: 300, cantidadItems: 2, estado: 'completado' },
//   { cliente: 'Juan', total: 50, cantidadItems: 1, estado: 'pendiente' },
//   { cliente: 'María', total: 525, cantidadItems: 3, estado: 'completado' }
// ]
