Etiquetas HTML HTML

Bienvenido al mundo del HTML, el lenguaje fundamental para crear la estructura de cualquier página web.

Etiquetas HTML Principales

Conoce las etiquetas fundamentales para estructurar tu sitio web

Estructura

  • <!DOCTYPE html> Define el tipo de documento
  • <html> Elemento raíz de una página HTML
  • <head> Contiene metadatos/información para el documento
  • <body> Contiene el contenido visible de la página
  • <header> Define el encabezado de una sección o página
  • <footer> Define el pie de página
  • <main> Especifica el contenido principal del documento

Texto

  • <h1> a <h6> Encabezados de diferentes niveles
  • <p> Define un párrafo
  • <span> Contenedor en línea para texto
  • <strong> Define texto importante (negrita)
  • <em> Define texto enfatizado (cursiva)
  • <br> Salto de línea
  • <hr> Línea horizontal (separador)

Listas

  • <ul> Lista desordenada
  • <ol> Lista ordenada
  • <li> Elemento de lista
  • <dl> Lista de definición
  • <dt> Término en una lista de definición
  • <dd> Descripción en una lista de definición

Enlaces

  • <a> Define un hipervínculo
  • <nav> Define un conjunto de enlaces de navegación
  • <link> Define relación entre documento y recurso externo

Multimedia

  • <img> Define una imagen
  • <video> Inserta contenido de video
  • <audio> Inserta contenido de audio
  • <source> Define recursos para elementos multimedia
  • <figure> Especifica contenido independiente
  • <figcaption> Define una leyenda para una figura

Formularios

  • <form> Define un formulario HTML
  • <input> Define un campo de entrada
  • <textarea> Define un área de texto multilínea
  • <button> Define un botón
  • <select> Define una lista desplegable
  • <option> Define una opción en una lista desplegable
  • <label> Define una etiqueta para un elemento de formulario
