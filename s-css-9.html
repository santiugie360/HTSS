CSS Flexbox

Flexbox es un modelo de diseño CSS que permite crear layouts flexibles y responsivos de manera eficiente, distribuyendo el espacio y alineando elementos con facilidad.

¿Qué es Flexbox?

Modelo de Diseño Unidimensional

Flexbox es un modelo de diseño CSS que trabaja en una sola dimensión a la vez (fila o columna). Permite distribuir el espacio disponible entre los elementos de un contenedor y controlar su alineación.

A diferencia de los layouts tradicionales basados en bloques, Flexbox está diseñado específicamente para crear interfaces de usuario modernas y adaptables.

Contenedor y Elementos

Flexbox funciona con dos tipos de entidades: el contenedor flex (flex container) que contiene los elementos, y los elementos flex (flex items) que son los hijos directos del contenedor.

Al aplicar display: flex a un elemento, este se convierte en un contenedor flex y sus hijos se convierten automáticamente en elementos flex.

Eje Principal y Eje Cruzado

Flexbox trabaja con dos ejes: el eje principal (main axis) definido por flex-direction, y el eje cruzado (cross axis) perpendicular al principal.

Entender estos ejes es fundamental para dominar Flexbox, ya que propiedades como justify-content y align-items trabajan en ejes diferentes.

Diseño Responsivo

Flexbox facilita enormemente la creación de diseños responsivos. Los elementos pueden crecer, encogerse y reorganizarse automáticamente según el espacio disponible.

Con propiedades como flex-wrap y flex-grow, puedes crear layouts que se adaptan perfectamente a diferentes tamaños de pantalla sin media queries complejos.

Propiedades Principales

Conoce las propiedades esenciales de Flexbox

Propiedades del Contenedor

display

.container {
  display: flex;
}

Define un contenedor flex. Todos sus hijos directos se convierten en elementos flex.

flex-direction

.container {
  flex-direction: row | column | row-reverse | column-reverse;
}

Establece la dirección del eje principal (horizontal o vertical).

justify-content

.container {
  justify-content: flex-start | center | space-between | space-around;
}

Alinea los elementos a lo largo del eje principal.

align-items

.container {
  align-items: stretch | center | flex-start | flex-end;
}

Alinea los elementos a lo largo del eje cruzado.

flex-wrap

.container {
  flex-wrap: nowrap | wrap | wrap-reverse;
}

Controla si los elementos deben envolverse en múltiples líneas.

gap

.container {
  gap: 20px;
}

Define el espacio entre elementos flex.

Propiedades de los Elementos

flex-grow

.item {
  flex-grow: 1;
}

Define la capacidad del elemento para crecer si es necesario.

flex-shrink

.item {
  flex-shrink: 1;
}

Define la capacidad del elemento para encogerse si es necesario.

flex-basis

.item {
  flex-basis: 200px | auto;
}

Define el tamaño base del elemento antes de distribuir el espacio restante.

flex (abreviado)

.item {
  flex: 1 1 200px;
  /* grow shrink basis */
}

Forma abreviada para flex-grow, flex-shrink y flex-basis.

align-self

.item {
  align-self: auto | center | flex-start | flex-end;
}

Permite sobrescribir la alineación para un elemento individual.

order

.item {
  order: 2;
}

Controla el orden de los elementos. Por defecto es 0.
