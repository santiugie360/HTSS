Diseño con Flexbox

Aprende a crear layouts flexibles y responsivos con el módulo Flexbox de CSS

Introducción a Flexbox

Flexbox (Flexible Box Layout) es un modelo de diseño unidimensional que permite crear layouts flexibles y distribuir el espacio entre elementos de manera eficiente, incluso cuando sus tamaños son desconocidos o dinámicos.

¿Qué es Flexbox?

Flexbox es un modelo de layout diseñado para proporcionar una forma más eficiente de organizar, alinear y distribuir el espacio entre los elementos de un contenedor, incluso cuando su tamaño es desconocido o dinámico.

  • Diseñado específicamente para crear layouts
  • Distribución unidimensional (filas o columnas)
  • Control sobre alineación, orden y tamaño
  • Adaptable a diferentes tamaños de pantalla

Conceptos Básicos

Flexbox se basa en dos tipos de elementos y dos ejes principales:

  • Contenedor flex: El elemento padre con display: flex
  • Elementos flex: Los hijos directos del contenedor flex
  • Eje principal: Dirección principal del layout (horizontal o vertical)
  • Eje transversal: Perpendicular al eje principal

Ventajas de Flexbox

Flexibilidad

Los elementos pueden crecer o encogerse para adaptarse al espacio disponible, facilitando diseños responsivos.

Alineación

Control preciso sobre la alineación de elementos, tanto horizontal como verticalmente, sin necesidad de hacks.

Orden

Permite cambiar el orden visual de los elementos sin modificar el HTML, mejorando la accesibilidad y SEO.

Propiedades del Contenedor Flex

El contenedor flex es el elemento padre que tiene display: flex o display: inline-flex. Sus propiedades controlan la disposición de los elementos flex (hijos).

display: flex

Esta propiedad define un contenedor flex, transformando todos sus hijos directos en elementos flex.

.contenedor {
    display: flex;
    /* o */
    display: inline-flex; /* Para comportamiento inline */
}

Propiedades de los Elementos Flex

Los elementos flex son los hijos directos de un contenedor flex. Estas propiedades controlan cómo se comportan individualmente dentro del contenedor.

flex-grow

Define la capacidad de un elemento flex para crecer si es necesario. Acepta un valor numérico sin unidades que determina qué cantidad de espacio disponible dentro del contenedor debe ocupar el elemento.

.elemento {
    flex-grow: 0; /* Valor predeterminado */
    /* Otros valores */
    flex-grow: 1; /* Crecerá */
    flex-grow: 2; /* Crecerá el doble que flex-grow: 1 */
}

flex-shrink

Define la capacidad de un elemento flex para encogerse si es necesario. Funciona de manera similar a flex-grow, pero en la dirección opuesta.

.elemento {
    flex-shrink: 1; /* Valor predeterminado */
    /* Otros valores */
    flex-shrink: 0; /* No se encogerá */
    flex-shrink: 2; /* Se encogerá el doble que flex-shrink: 1 */
}

flex-basis

Define el tamaño inicial de un elemento flex antes de que se apliquen flex-grow o flex-shrink. Puede ser una longitud (px, em, etc.) o un porcentaje.

.elemento {
    flex-basis: auto; /* Valor predeterminado */
    /* Otros valores */
    flex-basis: 0;
    flex-basis: 100px;
    flex-basis: 25%;
}

flex (shorthand)

La propiedad abreviada flex combina flex-grow, flex-shrink y flex-basis en una sola declaración.

.elemento {
    flex: 0 1 auto; /* Valor predeterminado (grow shrink basis) */
    /* Valores comunes */
    flex: 1; /* Equivale a flex: 1 1 0% */
    flex: auto; /* Equivale a flex: 1 1 auto */
    flex: none; /* Equivale a flex: 0 0 auto */
}

align-self

Permite sobrescribir la alineación definida por align-items para elementos flex individuales.

.elemento {
    align-self: auto; /* Valor predeterminado (hereda de align-items) */
    /* Otros valores */
    align-self: flex-start;
    align-self: flex-end;
    align-self: center;
    align-self: stretch;
}

order

Controla el orden en que aparecen los elementos flex, independientemente de su orden en el HTML.

.elemento {
    order: 0; /* Valor predeterminado */
    /* Otros valores */
    order: 1; /* Aparecerá después de los elementos con order menor */
    order: -1; /* Aparecerá antes de los elementos con order mayor */
}
