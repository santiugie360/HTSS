CSS @keyframes

Las animaciones CSS con @keyframes te permiten crear movimientos y transiciones complejas, dando vida a tus páginas web de forma elegante y eficiente.

¿Qué son los @keyframes?

Definición de Animaciones

@keyframes es una regla CSS que permite definir los estilos que un elemento debe tener en diferentes momentos durante una animación. En lugar de solo tener un estado inicial y final, puedes especificar múltiples estados intermedios.

Cada fotograma clave (keyframe) representa un punto específico en el tiempo de la animación, permitiendo crear secuencias complejas y fluidas.

Control de Tiempo

Puedes especificar estados usando porcentajes (0%, 50%, 100%) o las palabras clave "from" (0%) y "to" (100%). Esto te da control preciso sobre cuándo ocurre cada cambio en la animación.

La propiedad animation permite controlar la duración, el timing, el retraso, la repetición y la dirección de las animaciones definidas con @keyframes.

Propiedades Animables

Puedes animar casi cualquier propiedad CSS: posición (transform), tamaño (width/height), color (background-color, color), opacidad (opacity), y muchas más.

Las propiedades transform y opacity son especialmente eficientes ya que el navegador puede optimizar su animación usando aceleración por hardware.

Reutilización

Una vez definida, una animación @keyframes puede aplicarse a múltiples elementos en tu página. Esto promueve la reutilización de código y mantiene tus estilos organizados.

Puedes combinar múltiples animaciones en un solo elemento, crear bibliotecas de animaciones reutilizables y mantener tu código CSS limpio y mantenible.

Sintaxis Básica

Aprende cómo estructurar tus animaciones con @keyframes

Estructura Básica

/* Definir la animación */
@keyframes nombreAnimacion {
  from {
    opacity: 0;
    transform: translateX(-100px);
  }
  
  to {
    opacity: 1;
    transform: translateX(0);
  }
}

/* Aplicar la animación */
.elemento {
  animation: nombreAnimacion 1s ease-in-out;
}

Explicación: La regla @keyframes define una animación llamada "nombreAnimacion". Los estados "from" y "to" representan el inicio (0%) y el final (100%) de la animación.

Usando Porcentajes

@keyframes rebote {
  0% {
    transform: translateY(0);
  }
  
  25% {
    transform: translateY(-30px);
  }
  
  50% {
    transform: translateY(0);
  }
  
  75% {
    transform: translateY(-15px);
  }
  
  100% {
    transform: translateY(0);
  }
}

Explicación: Los porcentajes permiten definir estados intermedios en la animación, creando efectos más complejos como este rebote.

Propiedades de Animation

.elemento {
  /* Forma abreviada */
  animation: nombre 2s ease-in-out 0.5s infinite alternate;
  
  /* Forma expandida */
  animation-name: nombre;
  animation-duration: 2s;
  animation-timing-function: ease-in-out;
  animation-delay: 0.5s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
  animation-fill-mode: forwards;
}

animation-duration: Duración total de la animación

animation-timing-function: Curva de aceleración (ease, linear, ease-in, ease-out)

animation-delay: Tiempo de espera antes de iniciar

animation-iteration-count: Número de repeticiones (número o infinite)

animation-direction: Dirección (normal, reverse, alternate, alternate-reverse)

animation-fill-mode: Estilos antes/después (none, forwards, backwards, both)

Ejemplos Prácticos

Fade In

@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

.elemento {
  animation: fadeIn 2s ease-in;
}

Deslizamiento

@keyframes slide {
  from {
    transform: translateX(-100px);
  }
  to {
    transform: translateX(0);
  }
}

.elemento {
  animation: slide 1s ease-out;
}

Rotación

@keyframes rotate {
  from {
    transform: rotate(0deg);
  }
  to {
    transform: rotate(360deg);
  }
}

.elemento {
  animation: rotate 2s linear infinite;
}

Escala Pulsante

@keyframes pulse {
  0%, 100% {
    transform: scale(1);
  }
  50% {
    transform: scale(1.2);
  }
}

.elemento {
  animation: pulse 1.5s ease-in-out infinite;
}

Rebote

@keyframes bounce {
  0%, 100% {
    transform: translateY(0);
  }
  50% {
    transform: translateY(-30px);
  }
}

.elemento {
  animation: bounce 1s ease infinite;
}

Cambio de Color

@keyframes colorChange {
  0% {
    background-color: #8b5cf6;
  }
  50% {
    background-color: #ec4899;
  }
  100% {
    background-color: #3b82f6;
  }
}

.elemento {
  animation: colorChange 3s infinite;
}

Consejos

  • • Experimenta con diferentes combinaciones de valores para crear efectos únicos
  • • La función "ease-in-out" generalmente produce animaciones más naturales
  • • Usa "alternate" para crear efectos de ida y vuelta automáticos
  • • Las transformaciones (transform) son más eficientes que animar posición directamente
  • • Copia el código generado y úsalo en tus propios proyectos
