CSS Grid

CSS Grid es el sistema de diseño bidimensional más poderoso de CSS, permitiendo crear layouts complejos con filas y columnas de manera elegante y eficiente.

¿Qué es CSS Grid?

Sistema Bidimensional

A diferencia de Flexbox que trabaja en una dimensión, CSS Grid maneja tanto filas como columnas simultáneamente, permitiendo crear layouts complejos de forma más natural.

Grid divide el espacio en celdas, organizadas en filas y columnas, donde puedes colocar elementos de manera precisa y crear diseños sofisticados con menos código.

Áreas de Grid

Grid permite nombrar áreas de tu layout con grid-template-areas, haciendo el código más legible y fácil de mantener.

Puedes visualizar tu layout directamente en el CSS, creando una representación ASCII que muestra exactamente cómo se organizan los elementos en la página.

Unidades Fraccionales

La unidad fr (fracción) distribuye el espacio disponible proporcionalmente. 1fr toma una fracción del espacio disponible.

Esta unidad flexible hace que sea fácil crear layouts que se adaptan al tamaño del contenedor, sin necesidad de calcular porcentajes complejos.

Posicionamiento Automático

Grid puede posicionar elementos automáticamente en las celdas disponibles, o puedes especificar exactamente dónde debe ir cada elemento usando líneas de grid.

Los elementos pueden expandirse a través de múltiples filas o columnas usando grid-column y grid-row, creando layouts asimétricos con facilidad.

Propiedades Principales

Conoce las propiedades esenciales de CSS Grid

Propiedades del Contenedor

display

.container {
  display: grid;
}

Define un contenedor grid. Todos sus hijos directos se convierten en elementos grid.

grid-template-columns

.container {
  grid-template-columns: 1fr 2fr 1fr;
  grid-template-columns: repeat(3, 1fr);
  grid-template-columns: 200px auto 100px;
}

Define el ancho de las columnas del grid.

grid-template-rows

.container {
  grid-template-rows: 100px 200px;
  grid-template-rows: repeat(3, 150px);
}

Define la altura de las filas del grid.

gap

.container {
  gap: 20px;
  gap: 20px 10px; /* fila columna */
  row-gap: 20px;
  column-gap: 10px;
}

Define el espacio entre filas y columnas.

justify-items / align-items

.container {
  justify-items: start | center | end | stretch;
  align-items: start | center | end | stretch;
}

Alinea elementos dentro de sus celdas (horizontal y vertical).

grid-template-areas

.container {
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
}

Define áreas nombradas en el grid para posicionamiento intuitivo.

Propiedades de los Elementos

grid-column

.item {
  grid-column: 1 / 3; /* desde línea 1 a 3 */
  grid-column: span 2; /* ocupa 2 columnas */
  grid-column-start: 1;
  grid-column-end: 3;
}

Define qué columnas ocupa el elemento.

grid-row

.item {
  grid-row: 1 / 3; /* desde línea 1 a 3 */
  grid-row: span 2; /* ocupa 2 filas */
  grid-row-start: 1;
  grid-row-end: 3;
}

Define qué filas ocupa el elemento.

grid-area

.item {
  grid-area: header; /* usa área nombrada */
  grid-area: 1 / 1 / 2 / 4; /* row-start / col-start / row-end / col-end */
}

Asigna el elemento a un área nombrada o especifica todas las líneas.

justify-self / align-self

.item {
  justify-self: start | center | end | stretch;
  align-self: start | center | end | stretch;
}

Alinea un elemento individual dentro de su celda.

Funciones Útiles

/* repeat() - repite patrones */
grid-template-columns: repeat(4, 1fr);

/* minmax() - mínimo y máximo */
grid-template-columns: minmax(200px, 1fr);

/* auto-fit / auto-fill - responsivo */
grid-template-columns: 
  repeat(auto-fit, minmax(200px, 1fr));

Funciones que hacen Grid más poderoso y flexible.
