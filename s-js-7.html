Document Object Model

El DOM es una interfaz de programación que representa la estructura de un documento HTML como un árbol de objetos, permitiendo a JavaScript manipular dinámicamente el contenido, estructura y estilo de las páginas web.

¿Qué es el DOM?

Estructura de Árbol

El DOM representa el documento HTML como una estructura jerárquica de nodos en forma de árbol. Cada elemento HTML se convierte en un nodo, con el elemento <html> como raíz.

Esta estructura permite navegar fácilmente entre elementos padres, hijos y hermanos, facilitando la manipulación del contenido.

Interfaz de Programación

El DOM proporciona una API (Interfaz de Programación de Aplicaciones) que JavaScript puede usar para acceder y modificar el contenido, estructura y estilos de una página web.

Métodos como getElementById(), querySelector() y createElement() son parte de esta interfaz.

Manipulación Dinámica

Con el DOM, puedes cambiar el contenido de la página después de que se haya cargado: agregar o eliminar elementos, modificar atributos, cambiar estilos CSS y responder a eventos del usuario.

Esto permite crear aplicaciones web interactivas y dinámicas sin necesidad de recargar la página completa.

Eventos Interactivos

El DOM permite capturar y responder a eventos del usuario como clics, movimientos del mouse, pulsaciones de teclas y más usando addEventListener().

Esto es fundamental para crear experiencias de usuario interactivas y responsivas en aplicaciones web modernas.

Métodos Principales

Conoce los métodos más importantes para manipular el DOM

Métodos de Selección

getElementById()

const element = document.getElementById('miId');
console.log(element); // 
...

Selecciona un elemento por su atributo ID. Es el más rápido.

querySelector()

const element = document.querySelector('.mi-clase');
const button = document.querySelector('button[type="submit"]');

Selecciona el primer elemento que coincida con un selector CSS.

querySelectorAll()

const elements = document.querySelectorAll('.item');
elements.forEach(el => console.log(el));

Selecciona todos los elementos que coincidan con un selector CSS.

getElementsByClassName()

const elements = document.getElementsByClassName('mi-clase');
// Devuelve una HTMLCollection (colección viva)

Selecciona todos los elementos con una clase específica.

getElementsByTagName()

const divs = document.getElementsByTagName('div');
console.log(divs.length); // Cantidad de divs

Selecciona todos los elementos de un tipo específico.

Métodos de Manipulación

createElement()

const div = document.createElement('div');
div.textContent = 'Hola Mundo';

Crea un nuevo elemento HTML.

appendChild()

const parent = document.getElementById('contenedor');
parent.appendChild(div); // Agrega al final

Agrega un elemento hijo al final de un elemento padre.

removeChild()

const parent = element.parentNode;
parent.removeChild(element); // Elimina el elemento

Elimina un elemento hijo de su padre.

setAttribute() / getAttribute()

element.setAttribute('class', 'nueva-clase');
const valor = element.getAttribute('data-id');

Establecer u obtener atributos de elementos.

classList

element.classList.add('activo');
element.classList.remove('inactivo');
element.classList.toggle('visible');

Manipular clases CSS de un elemento fácilmente.

Propiedades y Eventos

innerHTML / textContent

// innerHTML - interpreta HTML
element.innerHTML = '<strong>Texto</strong>';

// textContent - solo texto
element.textContent = 'Solo texto';

Obtener o establecer el contenido de un elemento.

style

element.style.color = 'red';
element.style.backgroundColor = 'blue';
element.style.fontSize = '20px';

Modificar estilos CSS directamente desde JavaScript.

addEventListener()

const btn = document.querySelector('button');
btn.addEventListener('click', () => {
  console.log('¡Clic!');
});

Escuchar eventos del usuario o del navegador.

Navegación en el DOM

element.parentNode      // Padre
element.children        // Hijos
element.nextSibling     // Siguiente hermano
element.previousSibling // Hermano anterior

Navegar entre elementos relacionados.

insertBefore() / insertAdjacentHTML()

parent.insertBefore(newEl, refEl);

element.insertAdjacentHTML('beforeend', '<p>Nuevo</p>');

Insertar elementos en posiciones específicas.

cloneNode()

const clone = element.cloneNode(true);
// true = clonar con hijos
document.body.appendChild(clone);

Crear una copia de un elemento.

Ejemplos Prácticos

Seleccionar y Modificar

// Seleccionar elemento
const titulo = document.getElementById('titulo');

// Cambiar contenido
titulo.textContent = 'Nuevo Título';

// Cambiar estilo
titulo.style.color = 'blue';
titulo.style.fontSize = '32px';

// Agregar clase
titulo.classList.add('destacado');

Seleccionar elementos y modificar su contenido, estilos y clases.

Crear y Agregar Elementos

// Crear nuevo elemento
const parrafo = document.createElement('p');
parrafo.textContent = 'Este es un párrafo nuevo';
parrafo.classList.add('texto');

// Agregar al documento
const contenedor = document.querySelector('.contenedor');
contenedor.appendChild(parrafo);

// Crear con HTML
contenedor.innerHTML += '<div>Nuevo div</div>';

Crear nuevos elementos dinámicamente y agregarlos al documento.

Manejo de Eventos

// Evento de clic
const boton = document.querySelector('#miBoton');
boton.addEventListener('click', () => {
  alert('¡Botón clickeado!');
});

// Evento de input
const input = document.querySelector('input');
input.addEventListener('input', (e) => {
  console.log('Valor:', e.target.value);
});

Responder a interacciones del usuario con eventos.

Eliminar Elementos

// Método moderno
const elemento = document.querySelector('.eliminar');
elemento.remove();

// Método clásico
const parent = elemento.parentNode;
parent.removeChild(elemento);

// Eliminar todos los hijos
const contenedor = document.querySelector('.contenedor');
contenedor.innerHTML = '';

Diferentes formas de eliminar elementos del DOM.

Lista Dinámica Interactiva

// HTML: <ul id="lista"></ul> <input id="input"> <button id="agregar">Agregar</button>

const lista = document.getElementById('lista');
const input = document.getElementById('input');
const boton = document.getElementById('agregar');

boton.addEventListener('click', () => {
  const texto = input.value;
  
  if (texto) {
    // Crear elemento de lista
    const li = document.createElement('li');
    li.textContent = texto;
    
    // Botón eliminar
    const btnEliminar = document.createElement('button');
    btnEliminar.textContent = '❌';
    btnEliminar.addEventListener('click', () => li.remove());
    
    li.appendChild(btnEliminar);
    lista.appendChild(li);
    input.value = '';
  }
});

Ejemplo completo de una lista interactiva donde puedes agregar y eliminar elementos dinámicamente.
