CSS Modelo de Caja

El Modelo de Caja (Box Model) es uno de los conceptos fundamentales de CSS. Define cómo se calculan las dimensiones de los elementos HTML y cómo se espacian entre sí. Cada elemento es una "caja" rectangular compuesta por contenido, padding, border y margin.

¿Qué es el Modelo de Caja?

Estructura Rectangular

Cada elemento HTML se representa como una caja rectangular. Esta caja tiene cuatro áreas distintas: el contenido, el padding (relleno), el border (borde) y el margin (margen exterior).

Esta estructura permite controlar precisamente el espacio que ocupa cada elemento en la página.

Cálculo de Dimensiones

Por defecto, cuando defines width y height, solo estás definiendo el área del contenido. El tamaño total del elemento incluye padding, border y margin.

Fórmula: Total = width + padding + border + margin

Espaciado y Diseño

El modelo de caja es esencial para crear layouts y controlar el espaciado. El padding crea espacio interno, el border define límites visuales, y el margin crea espacio entre elementos.

Entender este modelo es clave para crear diseños precisos y responsive.

Box Sizing

La propiedad box-sizing controla cómo se calculan las dimensiones. Con border-box, width y height incluyen padding y border.

Esto hace que el dimensionamiento sea mucho más intuitivo y predecible.

Contenido (Content)

El área donde se muestra el contenido real del elemento (texto, imágenes, etc.). Se controla con width y height. 

.elemento {
  width: 200px;
  height: 100px;
}

Padding (Relleno)

Espacio transparente entre el contenido y el borde. Aumenta el tamaño total del elemento. Se puede definir individual o globalmente. 

.elemento {
  padding: 20px;
  /* padding-top, right, bottom, left */
}

Border (Borde)

Línea que rodea el padding y el contenido. Puede tener color, grosor y estilo. Aumenta las dimensiones totales del elemento. 

.elemento {
  border: 2px solid #9333ea;
  border-radius: 8px;
}

Margin (Margen)

Espacio transparente fuera del borde. Separa el elemento de otros elementos. No tiene color de fondo y puede ser negativo. 

.elemento {
  margin: 30px;
  /* margin-top, right, bottom, left */
}

Cálculo del Tamaño Total

Ejemplo Práctico:

.caja {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid #9333ea;
  margin: 15px;
}

Ancho Total:

200px (content) + 20px (padding-left) + 20px (padding-right) + 5px (border-left) + 5px (border-right) + 15px (margin-left) + 15px (margin-right)

= 280px

Fórmulas:

Ancho Total:

width + padding-left + padding-right + border-left + border-right + margin-left + margin-right

Alto Total:

height + padding-top + padding-bottom + border-top + border-bottom + margin-top + margin-bottom

Área Visible (sin margin):

width + padding + border

Propiedades CSS

Propiedades individuales para controlar cada componente del box model

Width & Height (Ancho y Alto)

Propiedades Básicas:

  • width - Ancho del contenido
  • height - Alto del contenido
  • max-width - Ancho máximo
  • min-width - Ancho mínimo
  • max-height - Alto máximo
  • min-height - Alto mínimo
/* Dimensiones fijas */
.caja-fija {
  width: 300px;
  height: 200px;
}

/* Dimensiones responsive */
.caja-responsive {
  width: 100%;
  max-width: 600px;
  min-height: 200px;
}

/* Dimensiones relativas */
.caja-viewport {
  width: 50vw; /* 50% del viewport width */
  height: 30vh; /* 30% del viewport height */
}

Padding (Relleno)

Formas de Definir:

/* Un valor (todos los lados) */
padding: 20px;

/* Dos valores (vertical | horizontal) */
padding: 10px 20px;

/* Tres valores (top | horizontal | bottom) */
padding: 10px 20px 15px;

/* Cuatro valores (top | right | bottom | left) */
padding: 10px 20px 15px 25px;

Propiedades Individuales:

/* Padding individual por lado */
.elemento {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 15px;
  padding-left: 25px;
}

/* Combinación */
.otro {
  padding: 20px;
  padding-left: 40px; /* Sobrescribe */
}

Border (Borde)

Propiedades de Border:

/* Shorthand (grosor | estilo | color) */
border: 2px solid #9333ea;

/* Propiedades separadas */
border-width: 2px;
border-style: solid;
border-color: #9333ea;

/* Bordes individuales */
border-top: 3px dashed #c084fc;
border-right: 1px solid #a855f7;
border-bottom: 2px dotted #7e22ce;
border-left: 5px double #6b21a8;

Estilos de Border:

solid - Línea sólida
dashed - Línea discontinua
dotted - Línea de puntos
double - Línea doble 
/* Border radius (esquinas redondeadas) */
border-radius: 8px;
border-radius: 50%; /* Círculo */

Margin (Margen)

Sintaxis Similar a Padding:

/* Un valor (todos los lados) */
margin: 20px;

/* Dos valores (vertical | horizontal) */
margin: 10px 20px;

/* Cuatro valores (top | right | bottom | left) */
margin: 10px 20px 15px 25px;

/* Centrar horizontalmente */
margin: 0 auto;

/* Márgenes negativos */
margin-top: -10px; /* Superpone elementos */

Margin Collapse:

Importante: Los márgenes verticales adyacentes se "colapsan" y toman el valor del margen mayor, no la suma. 

/* Elemento 1 */
.elemento1 {
  margin-bottom: 30px;
}

/* Elemento 2 */
.elemento2 {
  margin-top: 20px;
}

/* Espacio entre ellos: 30px (no 50px) */

Box Sizing Property

Controla cómo se calculan las dimensiones de los elementos

⚠️

content-box

(Valor por defecto)

Width y height solo aplican al contenido. Padding y border se AGREGAN al tamaño especificado. 

.caja {
  box-sizing: content-box; /* default */
  width: 200px;
  padding: 20px;
  border: 5px solid red;
}

/* Ancho total renderizado: */
/* 200px + 20px + 20px + 5px + 5px = 250px */

Problema: Si especificas width: 200px, el elemento será más ancho debido al padding y border. Esto puede romper layouts.

✅

border-box

(Recomendado)

Width y height INCLUYEN padding y border. El contenido se ajusta automáticamente. 

.caja {
  box-sizing: border-box; /* ✅ mejor */
  width: 200px;
  padding: 20px;
  border: 5px solid green;
}

/* Ancho total renderizado: */
/* 200px (incluye todo) */

Ventaja: El elemento siempre tendrá exactamente 200px de ancho, sin importar el padding o border. ¡Mucho más predecible!

Mejor Práctica (Reset Universal)

La mayoría de los desarrolladores aplican border-box a TODOS los elementos al inicio de su CSS. Esto hace que el dimensionamiento sea mucho más intuitivo.

Reset Completo:

/* Aplicar a TODOS los elementos */
*,
*::before,
*::after {
  box-sizing: border-box;
}

/* Alternativa más específica */
html {
  box-sizing: border-box;
}

*,
*::before,
*::after {
  box-sizing: inherit;
}

Beneficios:

  • ✓
    Dimensionamiento más predecible
  • ✓
    Layouts más fáciles de controlar
  • ✓
    No necesitas calcular manualmente
  • ✓
    Diseños responsive más sencillos
  • ✓
    Es el estándar moderno en CSS

Recomendación: Siempre usa este reset al inicio de tu CSS. Es considerado una mejor práctica en desarrollo web moderno.

Ejemplos Prácticos

Card de Producto

.product-card {
  width: 300px;
  padding: 20px;
  border: 2px solid #9333ea;
  border-radius: 12px;
  margin: 20px;
  box-sizing: border-box;
}

Card con dimensiones fijas, padding uniforme y borde redondeado.

Botón con Espaciado

.button {
  padding: 12px 24px; /* vertical | horizontal */
  border: 2px solid #a855f7;
  border-radius: 8px;
  margin: 10px 5px;
  box-sizing: border-box;
}

Botón con padding asimétrico para mejor proporción visual.

Contenedor Centrado

.container {
  width: 90%;
  max-width: 1200px;
  margin: 0 auto; /* Centrar horizontalmente */
  padding: 0 20px;
  box-sizing: border-box;
}

Contenedor responsive que se centra y tiene ancho máximo.

Grid de Elementos

.grid-item {
  width: calc(33.333% - 20px); /* 3 columnas */
  padding: 15px;
  margin: 10px;
  border: 1px solid #7e22ce;
  box-sizing: border-box;
}

Elemento de grid con cálculo de ancho considerando márgenes.

Input con Ícono

.input-wrapper {
  position: relative;
}

.input {
  width: 100%;
  padding: 12px 40px 12px 15px; /* Espacio para ícono */
  border: 2px solid #9333ea;
  border-radius: 8px;
  box-sizing: border-box;
}

Input con padding asimétrico para acomodar un ícono interno.

Sidebar con Espaciado

.sidebar {
  width: 250px;
  height: 100vh;
  padding: 20px;
  border-right: 1px solid #6b21a8;
  box-sizing: border-box;
}

.sidebar-item {
  padding: 12px 16px;
  margin-bottom: 8px;
  border-radius: 6px;
}

Sidebar con altura completa y elementos con espaciado uniforme.
