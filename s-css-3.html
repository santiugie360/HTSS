Unidades de Medida y Colores en CSS

Aprende a utilizar las diferentes unidades de medida y a representar colores en tus estilos CSS

Introducción

Las unidades de medida y los colores son fundamentales en CSS para definir tamaños, espaciados y apariencia visual de los elementos. Elegir las unidades adecuadas es crucial para crear diseños responsivos y accesibles.

Unidades de Medida

Las unidades de medida en CSS se dividen en dos categorías principales: absolutas y relativas. Cada una tiene sus propios casos de uso y ventajas.

  • Unidades absolutas: px, pt, cm, mm, in
  • Unidades relativas: %, em, rem, vh, vw

Colores

CSS ofrece múltiples formas de representar colores, desde nombres predefinidos hasta valores numéricos en diferentes formatos.

  • Nombres de colores: red, blue, rebeccapurple
  • Hexadecimal: #FF0000, #0000FF
  • RGB/RGBA: rgb(255, 0, 0), rgba(0, 0, 255, 0.5)
  • HSL/HSLA: hsl(0, 100%, 50%), hsla(240, 100%, 50%, 0.5)

Unidades Absolutas

Las unidades absolutas tienen un valor fijo y no cambian según el contexto. Son útiles cuando se necesita precisión exacta, como en impresión o cuando el tamaño debe ser independiente de otros factores.

Píxeles (px)

La unidad más común en diseño web. Un píxel es un punto en la pantalla.

font-size: 16px;
margin: 10px;
width: 200px;

100px de ancho

Puntos (pt)

Utilizada principalmente para impresión. 1pt = 1/72 de pulgada.

font-size: 12pt;
margin: 6pt;
padding: 3pt;

Texto de 12pt

Texto de 16pt

Centímetros (cm)

Medida del mundo real. Útil para diseños que requieren precisión física.

width: 5cm;
height: 2cm;
margin: 0.5cm;

≈ 2.54cm (1 pulgada)

Milímetros (mm)

Similar a centímetros pero más preciso. 10mm = 1cm.

border-width: 2mm;
padding: 5mm;
margin: 10mm;

10mm × 10mm

Pulgadas (in)

Unidad de medida imperial. 1in = 96px = 2.54cm.

width: 1in;
height: 0.5in;
margin: 0.25in;

1in × 0.25in

Picas (pc)

Utilizada en tipografía. 1pc = 12pt = 1/6 de pulgada.

text-indent: 1pc;
margin-left: 2pc;
padding: 0.5pc;

Este párrafo tiene una sangría de 1 pica, que equivale a 12 puntos o 1/6 de pulgada.

Consideraciones sobre Unidades Absolutas

Ventajas:

  • Precisión exacta
  • Consistencia en diferentes contextos
  • Ideales para impresión
  • Fáciles de entender y visualizar

Desventajas:

  • No se adaptan a diferentes tamaños de pantalla
  • Pueden causar problemas de accesibilidad
  • No responden a las preferencias del usuario
  • Menos flexibles para diseños responsivos

Consejo: En diseño web moderno, se recomienda usar unidades absolutas solo para elementos que realmente necesiten un tamaño fijo, como bordes o pequeños detalles visuales.

Unidades Relativas

Las unidades relativas cambian su valor según el contexto, como el tamaño de la fuente del elemento padre, el tamaño de la ventana del navegador o el tamaño de la fuente raíz. Son fundamentales para crear diseños responsivos.

Porcentaje (%)

Relativo al tamaño del elemento padre.

width: 50%;
margin: 10%;
font-size: 120%;

50% del ancho del contenedor

em

Relativo al tamaño de fuente del elemento padre.

font-size: 1.2em;
margin: 1em;
padding: 0.5em;
Texto base (16px)
Texto 1.5em (24px)
Texto 1.5em (36px)

rem

Relativo al tamaño de fuente del elemento raíz (html).

font-size: 1.2rem;
margin: 2rem;
width: 10rem;
Texto base
Texto 1.5rem
También 1.5rem (no se acumula)

Viewport Width (vw)

1vw = 1% del ancho de la ventana del navegador.

width: 50vw;
font-size: 5vw;
margin-left: 10vw;

30vw de ancho

Viewport Height (vh)

1vh = 1% de la altura de la ventana del navegador.

height: 50vh;
min-height: 30vh;
margin-top: 5vh;
10vh de alto

Otras Unidades Viewport

vmin, vmax y otras unidades relativas al viewport.

/* 1% de la dimensión más pequeña */
width: 50vmin;

/* 1% de la dimensión más grande */
height: 30vmax;

10vmin

10vmax × 10vmin

Cuándo usar cada unidad relativa

Para tipografía:

  • rem: Para tamaños de fuente consistentes
  • em: Para espaciado relacionado con el tamaño de fuente
  • %: Para ajustes relativos al tamaño base

Para layout:

  • %: Para anchos relativos al contenedor
  • vw/vh: Para elementos relativos a la ventana
  • vmin/vmax: Para mantener proporciones

Buena práctica: Combina unidades relativas para crear diseños que se adapten a diferentes dispositivos y preferencias de usuario. Por ejemplo, usa rem para tipografía, % para layouts y vw/vh para elementos que deben escalarse con la ventana.

Representación de Colores

CSS ofrece múltiples formas de representar colores, cada una con sus propias ventajas y casos de uso. Entender estos formatos te permite tener un control preciso sobre la apariencia visual de tu sitio.

Nombres de Colores

CSS incluye 140+ nombres de colores predefinidos que puedes usar directamente.

color: red;
background-color: rebeccapurple;
border-color: gold;
red rebeccapurple gold

Hexadecimal

Formato de 6 dígitos que representa valores RGB en base 16. También existe la versión abreviada de 3 dígitos.

color: #FF0000;        /* Rojo */
background: #00FF00;    /* Verde */
border-color: #0000FF;  /* Azul */
color: #F00;            /* Rojo abreviado */
#FF0000 #00FF00 #0000FF

RGB / RGBA

Define colores usando valores de Rojo, Verde y Azul (0-255). RGBA añade un canal alfa para transparencia (0-1).

color: rgb(255, 0, 0);          /* Rojo */
background: rgba(0, 0, 255, 0.5); /* Azul 50% opaco */
rgb(255, 0, 0) rgb(0, 255, 0) rgba(0, 0, 255, 0.5)

HSL / HSLA

Define colores usando Tono (0-360), Saturación (0-100%) y Luminosidad (0-100%). HSLA añade transparencia.

color: hsl(0, 100%, 50%);          /* Rojo */
background: hsla(240, 100%, 50%, 0.5); /* Azul 50% opaco */
hsl(0, 100%, 50%) hsl(120, 100%, 50%) hsla(240, 100%, 50%, 0.5)

Comparación de Formatos de Color

Formato Ventajas Desventajas Mejor para
Nombres Fácil de recordar y usar Limitado a 140+ colores predefinidos Prototipos rápidos, colores básicos
Hexadecimal Compacto, ampliamente usado Difícil de interpretar a simple vista Uso general, compatibilidad
RGB/RGBA Intuitivo, soporta transparencia Más verboso que hexadecimal Cuando necesitas transparencia
HSL/HSLA Más intuitivo para ajustes de color Menos común, más verboso Generación de paletas, ajustes de color

Funciones Modernas de Color

CSS ha introducido nuevas funciones para trabajar con colores de manera más avanzada:

color-mix()

Mezcla dos colores en un espacio de color específico.

color: color-mix(in srgb, #ff0000 50%, #0000ff 50%);

color-contrast()

Selecciona el color con mejor contraste de una lista.

color: color-contrast(#background vs #fff, #000);
