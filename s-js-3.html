Operadores y Expresiones

Aprende a realizar operaciones y crear expresiones en JavaScript

Operadores Aritméticos

Los operadores aritméticos te permiten realizar operaciones matemáticas básicas como suma, resta, multiplicación y división. Son como las operaciones que aprendiste en la escuela, pero aplicadas a la programación.

+

Suma (+)

Suma dos valores o concatena strings.

let suma = 5 + 3;     // 8
let texto = "Hola " + "Mundo"; // "Hola Mundo"

Cuidado: si sumas un número y un string, JavaScript convertirá el número a string y los concatenará.

-

Resta (-)

Resta el segundo valor del primero.

let resta = 10 - 4;     // 6
let negativo = -5;      // Número negativo

También se usa como operador unario para convertir un valor a su versión negativa.

*

Multiplicación (*)

Multiplica dos valores.

let producto = 4 * 5;     // 20
let area = ancho * alto;  // Calcula área

Útil para cálculos matemáticos como áreas, volúmenes o escalas.

/

División (/)

Divide el primer valor entre el segundo.

let division = 20 / 4;     // 5
let decimal = 10 / 3;      // 3.3333...

A diferencia de otros lenguajes, en JavaScript la división siempre devuelve un número decimal (float), incluso si el resultado es un número entero.

%

Módulo (%)

Devuelve el resto de la división del primer valor entre el segundo.

let resto = 10 % 3;     // 1 (10 ÷ 3 = 3 con resto 1)
let esPar = numero % 2 === 0; // true si es par

Muy útil para determinar si un número es par o impar, o para operaciones cíclicas (como rotar entre 0-359 grados).

**

Exponenciación (**)

Eleva el primer valor a la potencia del segundo valor.

let cuadrado = 5 ** 2;     // 25 (5 al cuadrado)
let cubo = 2 ** 3;         // 8 (2 al cubo)

Es una alternativa más legible a Math.pow(base, exponente).

++/--

Incremento/Decremento

Incrementa (++) o decrementa (--) un valor en 1.

let contador = 5;
contador++;      // contador ahora es 6
contador--;      // contador ahora es 5 de nuevo

// Diferencia entre prefijo y sufijo
let a = 5;
let b = ++a;     // a = 6, b = 6 (incrementa primero)
let c = 5;
let d = c++;     // c = 6, d = 5 (incrementa después)

Pueden usarse como prefijo (++x) o sufijo (x++), con diferente comportamiento cuando se asignan a otra variable.

Precedencia de Operadores

Al igual que en matemáticas, JavaScript sigue un orden específico al evaluar expresiones con múltiples operadores. Esto se llama "precedencia de operadores".

Orden de evaluación

  1. Paréntesis ()
  2. Exponenciación **
  3. Multiplicación *, División /, Módulo %
  4. Suma +, Resta -

Si dos operadores tienen la misma precedencia, se evalúan de izquierda a derecha (excepto la exponenciación, que va de derecha a izquierda).

Ejemplos

// Sin paréntesis (sigue precedencia)
let resultado1 = 2 + 3 * 4;     // 14 (no 20)
// 3 * 4 = 12, luego 2 + 12 = 14

// Con paréntesis (cambia precedencia)
let resultado2 = (2 + 3) * 4;   // 20
// (2 + 3) = 5, luego 5 * 4 = 20

// Expresión compleja
let resultado3 = 2 ** 3 + 4 * 5 - 6 / 2;
// 2 ** 3 = 8, 4 * 5 = 20, 6 / 2 = 3
// 8 + 20 - 3 = 25

Cuando tengas dudas sobre el orden de evaluación, usa paréntesis para hacerlo más claro y evitar errores.

Consejo: Cuando una expresión es compleja, es mejor usar paréntesis para hacer explícito el orden de las operaciones, incluso si coincide con la precedencia por defecto. Esto hace que tu código sea más legible y menos propenso a errores.

Operadores de Comparación

Los operadores de comparación te permiten comparar dos valores y devuelven un valor booleano (true o false). Son fundamentales para tomar decisiones en tu código, como en las estructuras condicionales.

==

Igualdad (==)

Comprueba si dos valores son iguales, con conversión de tipos.

5 == 5;       // true
"5" == 5;     // true (convierte string a número)
true == 1;    // true (convierte booleano a número)
null == undefined; // true

JavaScript intenta convertir los valores a un tipo común antes de compararlos, lo que puede llevar a resultados inesperados.

===

Igualdad estricta (===)

Comprueba si dos valores son iguales, sin conversión de tipos.

5 === 5;       // true
"5" === 5;     // false (tipos diferentes)
true === 1;    // false (tipos diferentes)
null === undefined; // false

Es más seguro usar === en la mayoría de los casos para evitar conversiones inesperadas.

!=

Desigualdad (!=)

Comprueba si dos valores son diferentes, con conversión de tipos.

5 != 8;       // true
"5" != 5;     // false (son iguales después de conversión)
true != 0;    // true
null != undefined; // false (son iguales con conversión)

Es el opuesto del operador de igualdad (==).

!==

Desigualdad estricta (!==)

Comprueba si dos valores son diferentes, sin conversión de tipos.

5 !== 8;       // true
"5" !== 5;     // true (tipos diferentes)
true !== 1;    // true (tipos diferentes)
null !== undefined; // true

Es el opuesto del operador de igualdad estricta (===).

< >

Mayor/Menor que

Comparan si un valor es mayor o menor que otro.

5 > 3;        // true (5 es mayor que 3)
5 < 3;        // false (5 no es menor que 3)
5 >= 5;       // true (5 es mayor o igual a 5)
5 <= 4;       // false (5 no es menor o igual a 4)

También funcionan con strings, comparando alfabéticamente ("a" < "b" es true).

?

Operador ternario

Una forma corta de escribir una condición con tres operandos.

// condición ? valorSiVerdadero : valorSiFalso
let edad = 20;
let mensaje = edad >= 18 ? "Adulto" : "Menor";
// mensaje = "Adulto"

// Equivalente a:
let mensaje2;
if (edad >= 18) {
  mensaje2 = "Adulto";
} else {
  mensaje2 = "Menor";
}

Útil para asignaciones condicionales simples, pero no abuses de él para lógica compleja.

Comparaciones y Tipos de Datos

Las comparaciones en JavaScript pueden ser confusas debido a la conversión automática de tipos. Veamos algunos casos especiales:

Comparación de strings

// Comparación alfabética (basada en Unicode)
"a" < "b";         // true
"apple" < "banana"; // true

// Cuidado con mayúsculas y minúsculas
"A" < "a";         // true (mayúsculas vienen antes)

// Comparación de números como strings
"10" < "2";        // true (compara caracter por caracter)
// "1" < "2", por lo que "10" < "2"

Para comparar números almacenados como strings, conviértelos primero: Number("10") > Number("2")

Casos especiales

// null y undefined
null == undefined;  // true
null === undefined; // false

// NaN (Not a Number)
NaN == NaN;         // false (NaN no es igual a nada, ni a sí mismo)
isNaN(NaN);         // true (usa esta función para verificar NaN)

// Objetos
{} == {};           // false (compara referencias, no contenido)
let obj = {};
obj == obj;         // true (misma referencia)

Para comparar objetos por su contenido, necesitas comparar sus propiedades o usar bibliotecas como Lodash.

Buena práctica: Usa siempre === y !== en lugar de == y != para evitar sorpresas con la conversión automática de tipos. Solo usa == cuando realmente necesites la conversión de tipos (lo cual es raro).

Operadores Lógicos

Los operadores lógicos te permiten combinar múltiples condiciones y trabajar con valores booleanos. Son esenciales para crear lógica compleja en tus programas.

&&

AND lógico (&&)

Devuelve true solo si ambas condiciones son true.

true && true;    // true
true && false;   // false
false && true;   // false
false && false;  // false

// Ejemplo práctico
let edad = 25;
let tieneID = true;

if (edad >= 18 && tieneID) {
  console.log("Puede entrar al club");
}

Es como decir "ambas condiciones deben ser verdaderas". Si la primera condición es false, la segunda ni siquiera se evalúa (evaluación de cortocircuito).

||

OR lógico (||)

Devuelve true si al menos una de las condiciones es true.

true || true;    // true
true || false;   // true
false || true;   // true
false || false;  // false

// Ejemplo práctico
let esFeriado = false;
let esFinDeSemana = true;

if (esFeriado || esFinDeSemana) {
  console.log("No hay que trabajar");
}

Es como decir "al menos una condición debe ser verdadera". Si la primera condición es true, la segunda ni siquiera se evalúa (evaluación de cortocircuito).

!

NOT lógico (!)

Invierte el valor booleano, convirtiendo true en false y viceversa.

!true;     // false
!false;    // true
!!true;    // true (doble negación)

// Ejemplo práctico
let estaLogueado = false;

if (!estaLogueado) {
  console.log("Por favor, inicia sesión");
}

// Conversión a booleano
!!"Hola";  // true (string no vacío es truthy)
!!"";      // false (string vacío es falsy)
!!0;       // false (0 es falsy)
!!1;       // true (números distintos de 0 son truthy)

Útil para negar condiciones o convertir valores a booleanos con doble negación (!!).

Valores Truthy y Falsy

En JavaScript, cualquier valor puede ser evaluado en un contexto booleano, como en una condición if. Los valores se clasifican como "truthy" (evaluados como true) o "falsy" (evaluados como false).

Valores Falsy

Estos valores se evalúan como false:

  • false - El booleano false
  • 0 - El número cero
  • -0 - El número cero negativo
  • 0n - BigInt cero
  • "", '', `` - String vacío
  • null - Ausencia de valor
  • undefined - Valor no definido
  • NaN - Not a Number
if (0) {
  // Este código nunca se ejecuta
}

if ("") {
  // Este código nunca se ejecuta
}

Valores Truthy

Todo lo demás se evalúa como true, incluyendo:

  • true - El booleano true
  • Cualquier número distinto de 0 (incluso negativos)
  • Cualquier string no vacío, incluso "0" y "false"
  • Todos los objetos y arrays, incluso vacíos {}, []
  • Funciones
if ("Hola") {
  // Este código se ejecuta
}

if ([]) {
  // Este código se ejecuta (¡un array vacío es truthy!)
}

Operadores lógicos y cortocircuito

Los operadores && y || no solo devuelven valores booleanos, sino que pueden devolver el valor de uno de sus operandos:

// OR (||) devuelve el primer valor truthy o el último valor
let nombre = nombreUsuario || "Invitado";
// Si nombreUsuario es undefined o "", nombre será "Invitado"

// AND (&&) devuelve el primer valor falsy o el último valor
let resultado = isAdmin && eliminarUsuario();
// eliminarUsuario() solo se ejecuta si isAdmin es true

Esto se llama "evaluación de cortocircuito" y es muy útil para establecer valores por defecto o ejecutar código condicionalmente.

Consejo: Ten cuidado con los valores falsy inesperados. Por ejemplo, 0 es falsy, lo que puede causar problemas si estás verificando un número que podría ser cero legítimamente. En esos casos, usa comparaciones explícitas como if (valor !== undefined).

Operadores de Asignación

Los operadores de asignación te permiten asignar valores a variables. Además del operador básico =, JavaScript ofrece operadores compuestos que combinan una operación con la asignación.

=

Asignación básica (=)

Asigna el valor de la derecha a la variable de la izquierda.

let x = 10;
let nombre = "Juan";
let activo = true;

Es el operador de asignación más básico y común.

+=

Suma y asignación (+=)

Suma el valor de la derecha al valor actual y asigna el resultado.

let x = 5;
x += 3;  // Equivale a: x = x + 3;
// x ahora es 8

let texto = "Hola";
texto += " Mundo";  // texto ahora es "Hola Mundo"

Funciona tanto con números (suma) como con strings (concatenación).

-=

Resta y asignación (-=)

Resta el valor de la derecha al valor actual y asigna el resultado.

let x = 10;
x -= 4;  // Equivale a: x = x - 4;
// x ahora es 6

Útil para decrementar contadores o ajustar valores numéricos.

*=

Multiplicación y asignación (*=)

Multiplica el valor actual por el valor de la derecha y asigna el resultado.

let x = 3;
x *= 4;  // Equivale a: x = x * 4;
// x ahora es 12

Útil para escalas, factores o cálculos acumulativos.

/=

División y asignación (/=)

Divide el valor actual por el valor de la derecha y asigna el resultado.

let x = 15;
x /= 3;  // Equivale a: x = x / 3;
// x ahora es 5

Útil para divisiones proporcionales o reducciones de escala.

%=

Módulo y asignación (%=)

Calcula el resto de dividir el valor actual por el valor de la derecha y asigna el resultado.

let x = 17;
x %= 5;  // Equivale a: x = x % 5;
// x ahora es 2 (el resto de 17 ÷ 5)

Útil para operaciones cíclicas o para mantener un valor dentro de un rango.

**=

Exponenciación y asignación (**=)

Eleva el valor actual a la potencia del valor de la derecha y asigna el resultado.

let x = 2;
x **= 3;  // Equivale a: x = x ** 3;
// x ahora es 8 (2 al cubo)

Útil para cálculos exponenciales o geométricos.

??=

Asignación de coalescencia nula (??=)

Asigna el valor de la derecha solo si el valor actual es null o undefined.

let x;
x ??= 10;  // x era undefined, ahora es 10

let y = 5;
y ??= 20;  // y ya tenía un valor, sigue siendo 5

Perfecto para asignar valores por defecto solo cuando es necesario.

Asignación Desestructurante

La asignación desestructurante es una sintaxis especial que te permite extraer datos de arrays u objetos y asignarlos a variables de forma concisa.

Desestructuración de Arrays

// Asignación básica
const numeros = [1, 2, 3, 4, 5];
const [primero, segundo] = numeros;
console.log(primero);  // 1
console.log(segundo);  // 2

// Saltar elementos
const [a, , c] = numeros;
console.log(a, c);  // 1, 3

// Resto de elementos
const [x, y, ...resto] = numeros;
console.log(x, y);     // 1, 2
console.log(resto);    // [3, 4, 5]

// Valores por defecto
const [p = 10, q = 20] = [5];
console.log(p, q);     // 5, 20

Muy útil para trabajar con funciones que devuelven arrays o para extraer elementos específicos.

Desestructuración de Objetos

// Asignación básica
const persona = {
  nombre: "Ana",
  edad: 28,
  ciudad: "Madrid"
};

const { nombre, edad } = persona;
console.log(nombre);  // "Ana"
console.log(edad);    // 28

// Renombrar propiedades
const { nombre: nombreCompleto, ciudad: ubicacion } = persona;
console.log(nombreCompleto);  // "Ana"
console.log(ubicacion);       // "Madrid"

// Valores por defecto
const { profesion = "Desconocida" } = persona;
console.log(profesion);  // "Desconocida"

// Resto de propiedades
const { nombre: n, ...datos } = persona;
console.log(n);      // "Ana"
console.log(datos);  // { edad: 28, ciudad: "Madrid" }

Excelente para extraer propiedades específicas de objetos, especialmente en parámetros de funciones.

Casos de uso prácticos

// Intercambiar variables
let a = 5, b = 10;
[a, b] = [b, a];
console.log(a, b);  // 10, 5

// Parámetros de función
function mostrarInfo({ nombre, edad = 'No especificada' }) {
  console.log(`Nombre: ${nombre}, Edad: ${edad}`);
}
mostrarInfo({ nombre: "Carlos" });  // "Nombre: Carlos, Edad: No especificada"

// Devolver múltiples valores
function obtenerDimensiones() {
  return [800, 600];
}
const [ancho, alto] = obtenerDimensiones();
console.log(`Ancho: ${ancho}px, Alto: ${alto}px`);  // "Ancho: 800px, Alto: 600px"

Consejo: La desestructuración hace que tu código sea más limpio y legible, especialmente cuando trabajas con objetos complejos o APIs que devuelven múltiples valores. Es una característica moderna de JavaScript que vale la pena dominar.

Ejercicios para practicar

Aquí tienes algunos ejercicios sencillos para practicar lo que has aprendido:

Ejercicio 1: Calculadora de IMC

Escribe una expresión que calcule el Índice de Masa Corporal (IMC) usando la fórmula: peso (kg) / (altura (m) * altura (m))

Solución:
// Calculadora de IMC
let peso = 70; // en kilogramos
let altura = 1.75; // en metros

let imc = peso / (altura ** 2);
console.log(`Tu IMC es: ${imc.toFixed(2)}`);

Ejercicio 2: Verificar elegibilidad

Escribe una expresión que determine si una persona es elegible para un descuento especial (debe ser mayor de 65 años O menor de 18 años Y tener membresía).

Solución:
// Verificar elegibilidad para descuento
let edad = 16;
let tieneMembresía = true;

let esElegible = (edad > 65 || (edad < 18 && tieneMembresía));
console.log(`¿Es elegible para el descuento? ${esElegible}`);
