Etiquetas básicas de HTML

¿Qué son y cómo se usan las etiquetas fundamentales para crear páginas web?

Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. No es un lenguaje de programación, sino un lenguaje de marcado que define la estructura de tu contenido.

HTML usa "etiquetas" para definir diferentes elementos en una página web. Estas etiquetas le dicen al navegador cómo mostrar el contenido.

Fácil de aprender

HTML tiene una estructura simple y lógica que cualquier persona puede aprender, incluso sin experiencia previa en programación.

Universal

Todos los navegadores web entienden HTML, lo que significa que tu contenido será accesible para cualquier persona con acceso a internet.

Base de todo

HTML es el fundamento de cualquier sitio web. Incluso las páginas más complejas comienzan con una estructura HTML básica.

Estructura básica

<!DOCTYPE html>
<html>
  <head>
    <title>Título de la página</title>
  </head>
  <body>
    Contenido de la página
  </body>
</html>

Toda página HTML comienza con la declaración !DOCTYPE html que le dice al navegador que estamos usando HTML5.

La etiqueta html es el elemento raíz que contiene toda la página.

Dentro de head colocamos información sobre la página, como el título que aparece en la pestaña del navegador.

Todo lo que queremos mostrar en la página va dentro de body.

Etiquetas de texto

Encabezados

HTML tiene seis niveles de encabezados, desde h1 (el más importante) hasta h6 (el menos importante).

Esto es un h1

Esto es un h2

Esto es un h3

Párrafos y formato

La etiqueta p se usa para crear párrafos de texto.

Para dar énfasis al texto usamos strong o em para texto en cursiva.

Para crear un
salto de línea usamos br.

Enlaces

Los enlaces o hipervínculos son la base de la navegación web. Se crean con la etiqueta a.

<a href="https://www.ejemplo.com">Visita Ejemplo.com</a>

El atributo href especifica la dirección a la que apunta el enlace. Puede ser una URL externa, una página dentro de tu sitio, o incluso una sección específica de la página actual usando un identificador con #.

Listas

Listas ordenadas

Las listas ordenadas usan ol y cada elemento se define con li.

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

Listas no ordenadas

Las listas no ordenadas usan ul y cada elemento también se define con li.

  • Elemento
  • Otro elemento
  • Un elemento más

Imágenes

Las imágenes se insertan con la etiqueta img, que es una etiqueta de cierre automático (no necesita etiqueta de cierre).

<img src="ruta-de-la-imagen.jpg" alt="Descripción de la imagen">

El atributo src especifica la ruta de la imagen.

El atributo alt proporciona una descripción textual de la imagen, importante para la accesibilidad y SEO.
