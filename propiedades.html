Propiedades en CSS

Bienvenido al mundo del CSS, el lenguaje que da estilo y diseño a tus páginas web.

Propiedades CSS Principales

Conoce las propiedades fundamentales para dar estilo a tu sitio web

Selectores

  • elemento Selecciona todos los elementos de ese tipo
  • #id Selecciona el elemento con ese ID
  • .clase Selecciona todos los elementos con esa clase
  • elemento:pseudo-clase Selecciona elementos en un estado específico
  • elemento::pseudo-elemento Selecciona parte de un elemento
  • selector[atributo] Selecciona elementos con un atributo específico
  • selector1, selector2 Selecciona todos los elementos que coincidan

Layout y Posicionamiento

  • display Define cómo se muestra un elemento
  • position Especifica el método de posicionamiento
  • top, right, bottom, left Posiciona un elemento en relación a su contenedor
  • float Coloca un elemento a la izquierda o derecha
  • z-index Controla el orden de apilamiento
  • flex Establece cómo un elemento flexible crece o se encoge
  • grid Define un contenedor de cuadrícula

Tipografía

  • font-family Define la familia de fuentes
  • font-size Establece el tamaño de la fuente
  • font-weight Define el grosor de la fuente
  • line-height Establece la altura de línea
  • text-align Alinea el texto horizontalmente
  • text-decoration Añade decoración al texto (subrayado, tachado)
  • text-transform Controla la capitalización del texto

Colores y Fondos

  • color Define el color del texto
  • background-color Establece el color de fondo
  • background-image Define una imagen de fondo
  • background-size Especifica el tamaño de la imagen de fondo
  • background-position Define la posición inicial de la imagen de fondo
  • opacity Establece la transparencia de un elemento
  • gradient Crea transiciones suaves entre colores

Bordes y Efectos

  • border Define el borde de un elemento
  • border-radius Redondea las esquinas de un elemento
  • box-shadow Añade sombras a un elemento
  • text-shadow Añade sombras al texto
  • filter Aplica efectos visuales (blur, brightness, etc.)
  • backdrop-filter Aplica efectos al área detrás de un elemento
  • clip-path Crea formas complejas recortando un elemento

Animaciones y Transiciones

  • transition Define transiciones entre estados de un elemento
  • animation Aplica animaciones a elementos
  • @keyframes Define los pasos de una animación
  • transform Aplica transformaciones 2D o 3D a un elemento
  • translate Mueve un elemento horizontal o verticalmente
  • rotate Rota un elemento
  • scale Cambia el tamaño de un elemento
