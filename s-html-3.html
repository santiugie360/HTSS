HTML Elementos de Texto

Los elementos de texto son la base de cualquier página web. HTML proporciona una variedad de etiquetas para estructurar y dar formato al contenido textual: desde encabezados jerárquicos hasta listas organizadas, pasando por párrafos y formato de texto como negrita y cursiva.

¿Qué son los Elementos de Texto?

Estructura Semántica

Los elementos de texto en HTML no solo definen cómo se ve el contenido, sino también su significado y jerarquía. Los navegadores y motores de búsqueda utilizan estas etiquetas para entender la estructura del documento.

Usar las etiquetas correctas mejora la accesibilidad y el SEO de tu sitio web.

Jerarquía Visual

Los encabezados (<h1> a <h6>) crean una jerarquía visual y lógica. El <h1> es el más importante y debe usarse solo una vez por página.

Esta jerarquía ayuda a los usuarios a escanear y entender el contenido rápidamente.

Formato de Texto

HTML proporciona etiquetas para dar énfasis y formato al texto: <strong> para negrita (importancia), <em> para cursiva (énfasis), y más.

Cada etiqueta tiene un significado semántico que va más allá de su apariencia visual.

Listas Organizadas

Las listas permiten organizar información de forma estructurada. Puedes crear listas ordenadas (<ol>), desordenadas (<ul>) o de definición (<dl>).

Son perfectas para menús de navegación, características de productos, pasos de instrucciones, etc.

Ejemplo de Código:

<!-- Estructura jerárquica correcta -->
<h1>Título Principal de la Página</h1>

<h2>Sección 1</h2>
<p>Contenido de la sección...</p>

  <h3>Subsección 1.1</h3>
  <p>Contenido...</p>

  <h3>Subsección 1.2</h3>

Mejores Prácticas:

  • ✓
    Usa solo un <h1> por página
  • ✓
    Mantén la jerarquía en orden (no saltes niveles)
  • ✓
    No uses encabezados solo por su tamaño (usa CSS)
  • ✓
    Los encabezados deben describir el contenido que sigue
  • ✓
    Son importantes para SEO y accesibilidad

Importante: Los lectores de pantalla usan los encabezados para navegar. Una estructura correcta mejora la accesibilidad.

Párrafos <p>

Los párrafos son los bloques fundamentales de texto en HTML

Elemento <p>

La etiqueta <p> define un párrafo de texto. Los navegadores automáticamente agregan espacio antes y después de cada párrafo. 

<p>
  Este es un párrafo de texto. Puede contener 
  múltiples líneas y el navegador las ajustará 
  automáticamente según el ancho disponible.
</p>

<p>
  Este es otro párrafo. Los párrafos están 
  separados visualmente por espacio vertical.
</p>

Resultado visual:

Este es un párrafo de texto. Puede contener múltiples líneas y el navegador las ajustará automáticamente según el ancho disponible.

Este es otro párrafo. Los párrafos están separados visualmente por espacio vertical.

Saltos de Línea y Otros

Salto de Línea <br>

Crea un salto de línea sin iniciar un nuevo párrafo. 

<p>
  Primera línea<br>
  Segunda línea<br>
  Tercera línea
</p>

Línea Horizontal <hr>

Crea una línea horizontal divisoria temática. 

<p>Sección 1</p>
<hr>
<p>Sección 2</p>

Texto Preformateado <pre>

Preserva espacios y saltos de línea exactamente como están escritos. 

<pre>
  Texto    con    espacios
       y saltos
           preservados
</pre>

Formato de Texto HTML

Etiquetas para dar énfasis y significado al texto

<strong> - Texto Importante

Define texto con gran importancia (generalmente negrita)

<p>
  Esto es <strong>muy importante</strong>.
</p>
Esto es muy importante.

<em> - Texto con Énfasis

Define texto enfatizado (generalmente cursiva)

<p>
  Debes <em>realmente</em> leer esto.
</p>
Debes realmente leer esto.

<b> - Negrita Visual

Texto en negrita sin importancia semántica extra

<p>
  Palabras clave: <b>HTML</b>, <b>CSS</b>
</p>
Palabras clave: HTML, CSS

<i> - Cursiva Visual

Texto en cursiva sin énfasis semántico

<p>
  El término <i>smartphone</i> es común.
</p>
El término smartphone es común.

<mark> - Texto Resaltado

Texto marcado o resaltado con fondo amarillo

<p>
  <mark>Resalta</mark> información importante.
</p>
Resalta información importante.

<small> - Texto Pequeño

Texto de menor importancia (como notas al pie)

<p>
  Precio: $99 <small>IVA incluido</small>
</p>
Precio: $99 IVA incluido

<del> - Texto Eliminado

Texto que ha sido eliminado (tachado)

<p>
  Precio: <del>$150</del> $99
</p>
Precio: $150 $99

<ins> - Texto Insertado

Texto que ha sido añadido (subrayado)

<p>
  Texto <ins>insertado</ins> en el documento.
</p>
Texto insertado en el documento.

<sub> - Subíndice

Texto en subíndice (fórmulas químicas)

<p>
  H<sub>2</sub>O es agua
</p>
H2O es agua

<sup> - Superíndice

Texto en superíndice (potencias, notas al pie)

<p>
  E = mc<sup>2</sup>
</p>
E = mc2

<code> - Código

Fragmento de código de programación

<p>
  Usa <code>console.log()</code>
</p>
Usa console.log()

<abbr> - Abreviatura

Define una abreviatura o acrónimo

<p>
  <abbr title="Organización Mundial de la Salud">OMS</abbr>
</p>
OMS

Diferencias Semánticas

Etiqueta Significado Semántico Cuándo Usar
<strong> Importancia fuerte Advertencias, información crucial
<b> Sin significado extra Palabras clave, nombres de productos
<em> Énfasis de estrés Cambiar el significado de una frase
<i> Sin significado extra Términos técnicos, palabras extranjeras

Listas en HTML

Organiza información de manera estructurada con listas

•

Listas Desordenadas <ul>

Lista con viñetas (bullets)

Código:

<ul>
  <li>Manzanas</li>
  <li>Naranjas</li>
  <li>Plátanos</li>
</ul>

<!-- Con sublistas -->
<ul>
  <li>Frutas
    <ul>
      <li>Manzanas</li>
      <li>Naranjas</li>
    </ul>
  </li>
  <li>Verduras</li>
</ul>

Resultado:

  • Manzanas
  • Naranjas
  • Plátanos
  • Frutas
    • Manzanas
    • Naranjas
  • Verduras

Uso: Cuando el orden de los elementos NO es importante (ingredientes, características, etc.)

1.

Listas Ordenadas <ol>

Lista numerada secuencialmente

Código:

<ol>
  <li>Precalentar el horno</li>
  <li>Mezclar ingredientes</li>
  <li>Hornear por 20 minutos</li>
</ol>

<!-- Comenzar en número específico -->
<ol start="5">
  <li>Elemento 5</li>
  <li>Elemento 6</li>
</ol>

<!-- Orden inverso -->
<ol reversed>
  <li>Tercero</li>
  <li>Segundo</li>
  <li>Primero</li>
</ol>

Resultado:

  1. Precalentar el horno
  2. Mezclar ingredientes
  3. Hornear por 20 minutos
  1. Elemento 5
  2. Elemento 6
  1. Tercero
  2. Segundo
  3. Primero

Uso: Cuando el orden ES importante (pasos de receta, instrucciones, rankings)

📝

Listas de Definición <dl>

Lista de términos y sus definiciones

Código:

<dl>
  <dt>HTML</dt>
  <dd>Lenguaje de marcado para crear páginas web</dd>
  
  <dt>CSS</dt>
  <dd>Lenguaje para dar estilo a páginas web</dd>
  
  <dt>JavaScript</dt>
  <dd>Lenguaje de programación para interactividad</dd>
</dl>

Resultado:

HTML
Lenguaje de marcado para crear páginas web
CSS
Lenguaje para dar estilo a páginas web
JavaScript
Lenguaje de programación para interactividad

Uso: Glosarios, metadatos, pares clave-valor

Ejemplos Prácticos

Artículo de Blog

<article>
  <h1>Guía de HTML</h1>
  
  <h2>Introducción</h2>
  <p>
    HTML es el <strong>lenguaje fundamental</strong> 
    para crear páginas web.
  </p>
  
  <h2>Características</h2>
  <ul>
    <li>Fácil de aprender</li>
    <li>Amplio soporte</li>
    <li>Semántico</li>
  </ul>
</article>

Estructura típica de un artículo con encabezados, párrafos y lista.

Receta de Cocina

<article>
  <h1>Galletas de Chocolate</h1>
  
  <h2>Ingredientes</h2>
  <ul>
    <li>200g de harina</li>
    <li>150g de azúcar</li>
    <li>100g de chocolate</li>
  </ul>
  
  <h2>Instrucciones</h2>
  <ol>
    <li>Precalentar horno a 180°C</li>
    <li>Mezclar ingredientes</li>
    <li>Hornear 15 minutos</li>
  </ol>
</article>

Combina listas ordenadas y desordenadas para ingredientes e instrucciones.

Descripción de Producto

<div>
  <h2>Laptop Pro X</h2>
  
  <p>
    <strong>La mejor laptop</strong> para 
    profesionales. <del>$1,500</del> 
    <mark>$1,200</mark>
  </p>
  
  <h3>Especificaciones</h3>
  <dl>
    <dt>Procesador</dt>
    <dd>Intel i7</dd>
    
    <dt>RAM</dt>
    <dd>16GB</dd>
  </dl>
</div>

Usa formato de texto y listas de definición para especificaciones.

  <ul>
    <li><code>url</code> - <em>requerido</em></li>
    <li><code>options</code> - opcional</li>
  </ul>
</section>

Combina código, abreviaturas y énfasis para documentación clara.
