Codigo y conceptos en JavaScript

Bienvenido al mundo de JavaScript, el lenguaje que da interactividad y dinamismo a tus páginas web.

Conceptos JavaScript Principales

Conoce los fundamentos para dar vida y funcionalidad a tu sitio web

Fundamentos

  • var, let, const Declaración de variables
  • String, Number, Boolean Tipos de datos primitivos
  • null, undefined Valores especiales
  • +, -, *, /, % Operadores aritméticos
  • ==, ===, !=, !== Operadores de comparación
  • &&, ||, ! Operadores lógicos
  • typeof, instanceof Operadores de tipo

Estructuras de Control

  • if...else Condicional básico
  • switch Condicional múltiple
  • for Bucle con contador
  • while, do...while Bucles con condición
  • for...of Iteración sobre elementos iterables
  • for...in Iteración sobre propiedades de objetos
  • break, continue Control de flujo en bucles

Funciones

  • function declaration Declaración de función
  • function expression Expresión de función
  • arrow functions Funciones flecha
  • parameters, arguments Parámetros y argumentos
  • return Valor de retorno
  • callback Funciones como argumentos
  • closure Funciones que recuerdan su entorno

Objetos y Arrays

  • {}, new Object() Creación de objetos
  • [], new Array() Creación de arrays
  • object.property Acceso a propiedades
  • array[index] Acceso a elementos de array
  • push, pop, shift, unshift Métodos de array para añadir/eliminar
  • map, filter, reduce Métodos de array para transformar
  • Object.keys, Object.values Métodos para trabajar con objetos

DOM y Eventos

  • document.getElementById() Seleccionar elemento por ID
  • querySelector, querySelectorAll Seleccionar elementos con selectores CSS
  • element.innerHTML Modificar el contenido HTML
  • element.style Modificar estilos CSS
  • addEventListener() Registrar manejadores de eventos
  • click, submit, keydown Tipos de eventos comunes
  • event.preventDefault() Prevenir comportamiento por defecto

Asincronía

  • setTimeout, setInterval Temporizadores
  • Promise Objeto para operaciones asíncronas
  • .then(), .catch() Manejo de promesas
  • async/await Sintaxis para código asíncrono
  • fetch() API para realizar peticiones HTTP
  • try/catch Manejo de errores
  • JSON.parse(), JSON.stringify() Conversión de datos JSON
