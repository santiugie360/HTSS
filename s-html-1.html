Introducción a HTML

El lenguaje fundamental para crear la estructura de cualquier página web

¿Qué es HTML?

HTML (HyperText Markup Language) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.

Lenguaje de Marcado

HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido. HTML consiste en una serie de elementos que usarás para encerrar o envolver diferentes partes del contenido para que aparezcan o actúen de cierta manera.

Las etiquetas HTML pueden convertir una palabra o imagen en un hipervínculo, pueden poner palabras en cursiva, pueden aumentar el tamaño de la fuente y mucho más.

Bloques de Construcción

HTML proporciona los bloques de construcción básicos para la web. Define la estructura y el significado del contenido, pero no su apariencia. Para definir la apariencia y el diseño, se utilizan CSS (Cascading Style Sheets) y JavaScript para la interactividad.

Cada página web que visitas está construida con HTML, incluso las más complejas y dinámicas. HTML es la base sobre la que se construye toda la web.

Historia y Evolución

Desde su creación hasta la actualidad, HTML ha evolucionado significativamente

1991: HTML 1.0

Tim Berners-Lee crea HTML como un lenguaje para compartir documentos científicos. Incluía 18 elementos básicos.

1
2

1995: HTML 2.0

Primera especificación estándar de HTML. Introdujo formularios y soporte para gráficos.

1997: HTML 4.0

Introdujo hojas de estilo (CSS), scripts y marcos. Separó la estructura del estilo.

3
4

2000: XHTML

Reformulación de HTML como XML. Reglas más estrictas para escribir código.

2014: HTML5

Versión actual. Introdujo elementos semánticos, soporte para audio/video, canvas para gráficos y APIs para aplicaciones web.

Estructura Básica

Todo documento HTML sigue una estructura fundamental

Anatomía de un Documento HTML

1

Declaración DOCTYPE

Indica al navegador qué versión de HTML estás utilizando. Para HTML5, simplemente se usa <!DOCTYPE html>

2

Elemento HTML

El elemento raíz que envuelve todo el contenido de la página: <html>

3

Cabecera (Head)

Contiene metadatos, enlaces a CSS, título de la página y otros elementos no visibles: <head>

4

Cuerpo (Body)

Contiene todo el contenido visible de la página como texto, imágenes, enlaces, etc.: <body>

Ejemplo de Código


<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Mi Primera Página Web</title>
  <link rel="stylesheet" href="estilos.css">
</head>
<body>
  <header>
    <h1>Bienvenido a mi sitio web</h1>
    <nav>
      <ul>
        <li><a href="#">Inicio</a></li>
        <li><a href="#">Acerca de</a></li>
        <li><a href="#">Contacto</a></li>
      </ul>
    </nav>
  </header>

  <main>
    <section>
      <h2>Acerca de nosotros</h2>
      <p>Somos una empresa dedicada a...</p>
    </section>

    <section>
      <h2>Nuestros servicios</h2>
      <ul>
        <li>Servicio 1</li>
        <li>Servicio 2</li>
        <li>Servicio 3</li>
      </ul>
    </section>
  </main>

  <footer>
    <p>© 2023 Mi Empresa. Todos los derechos reservados.</p>
  </footer>
</body>
</html>

Importancia de HTML

¿Por qué HTML es fundamental en el desarrollo web?

Accesibilidad

Un HTML bien estructurado permite que las personas con discapacidades puedan acceder al contenido web mediante tecnologías de asistencia como lectores de pantalla.

SEO

Los motores de búsqueda utilizan el HTML para entender el contenido de una página. Un HTML semántico y bien estructurado mejora el posicionamiento en buscadores.

Rendimiento

Un HTML limpio y optimizado contribuye a tiempos de carga más rápidos, lo que mejora la experiencia del usuario y el rendimiento general del sitio.

Compatibilidad

HTML es compatible con todos los navegadores y dispositivos, lo que garantiza que tu contenido sea accesible para todos los usuarios, independientemente de cómo accedan a la web.

Escalabilidad

HTML proporciona una base sólida para construir sitios web que pueden crecer y evolucionar con el tiempo, permitiendo integrar nuevas tecnologías y funcionalidades.

Semántica

HTML5 introdujo elementos semánticos que dan significado al contenido, lo que mejora la estructura, la accesibilidad y la comprensión del propósito de cada parte de la página.
