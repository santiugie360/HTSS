JavaScript Objects

Los objetos son colecciones de pares clave-valor que permiten almacenar datos relacionados y funcionalidad en una única estructura. Son la base de la programación orientada a objetos en JavaScript y fundamentales para organizar código complejo.

¿Qué son los Objetos?

Colección de Propiedades

Un objeto es una colección de propiedades, donde cada propiedad es un par clave-valor. Las claves son siempre strings (o Symbols), y los valores pueden ser de cualquier tipo: números, strings, arrays, funciones u otros objetos.

Por ejemplo: { nombre: "Juan", edad: 25 } tiene dos propiedades con sus respectivos valores.

Referencia, no Copia

Los objetos se pasan por referencia, no por valor. Esto significa que cuando asignas un objeto a una variable, estás creando una referencia al objeto original, no una copia.

Si modificas el objeto a través de cualquier referencia, todos los lugares que apunten a ese objeto verán los cambios. Para copiar un objeto necesitas métodos específicos.

Métodos y Comportamiento

Los objetos pueden contener funciones como valores, llamadas métodos. Estos métodos permiten que los objetos tengan comportamiento además de datos, siendo la base de la programación orientada a objetos.

Un método puede acceder a otras propiedades del mismo objeto usando la palabra clave this, creando objetos autónomos y funcionales.

Dinámicos y Flexibles

Los objetos en JavaScript son dinámicos: puedes agregar, modificar o eliminar propiedades en cualquier momento después de su creación, proporcionando gran flexibilidad.

Puedes acceder a las propiedades usando notación de punto (obj.propiedad) o notación de corchetes (obj['propiedad']).

Código JavaScript

const usuario = {
  nombre: "Ana García",
  edad: 28,
  profesion: "Desarrolladora",
  activo: true,
  saludar: function() {
    return `Hola, soy ${this.nombre}`;
  }
};

// Acceder a propiedades
console.log(usuario.nombre);      // "Ana García"
console.log(usuario["edad"]);    // 28

// Llamar a un método
console.log(usuario.saludar()); // "Hola, soy Ana García"

// Agregar nueva propiedad
usuario.email = "ana@email.com";

// Modificar propiedad
usuario.edad = 29;

// Eliminar propiedad
delete usuario.activo;

Métodos de Object

Funciones integradas para trabajar con objetos

Object.keys()

const persona = {
  nombre: "Carlos",
  edad: 30,
  ciudad: "Madrid"
};

const claves = Object.keys(persona);
console.log(claves);
// ["nombre", "edad", "ciudad"]

Devuelve un array con las claves (nombres de propiedades) del objeto.

Object.values()

const persona = {
  nombre: "Carlos",
  edad: 30,
  ciudad: "Madrid"
};

const valores = Object.values(persona);
console.log(valores);
// ["Carlos", 30, "Madrid"]

Devuelve un array con los valores de las propiedades del objeto.

Object.entries()

const persona = {
  nombre: "Carlos",
  edad: 30
};

const entradas = Object.entries(persona);
console.log(entradas);
// [["nombre", "Carlos"], ["edad", 30]]

// Útil para iterar
entradas.forEach(([clave, valor]) => {
  console.log(`${clave}: ${valor}`);
});

Devuelve un array de arrays [clave, valor] con todas las propiedades.

Object.assign()

const obj1 = { a: 1, b: 2 };
const obj2 = { c: 3, d: 4 };

const combinado = Object.assign({}, obj1, obj2);
console.log(combinado);
// { a: 1, b: 2, c: 3, d: 4 }

// También funciona con spread operator
const combinado2 = { ...obj1, ...obj2 };

Copia propiedades de uno o más objetos a un objeto destino.

Object.freeze() / seal()

const obj = { a: 1 };

// freeze - no permite modificar
Object.freeze(obj);
obj.a = 2;          // No tiene efecto
obj.b = 3;          // No se puede agregar

// seal - permite modificar, no agregar/eliminar
const obj2 = { x: 1 };
Object.seal(obj2);
obj2.x = 2;         // ✓ Funciona
obj2.y = 3;         // ✗ No se puede agregar

Controlar la mutabilidad de los objetos (freeze: inmutable, seal: semi-mutable).

hasOwnProperty()

const persona = {
  nombre: "Ana",
  edad: 25
};

console.log(persona.hasOwnProperty('nombre'));
// true

console.log(persona.hasOwnProperty('ciudad'));
// false

// Útil para verificar antes de acceder
if (persona.hasOwnProperty('email')) {
  console.log(persona.email);
}

Verifica si el objeto tiene una propiedad específica (no heredada).

Ejemplos Prácticos

Crear y Manipular Objetos

// Literal de objeto
const libro = {
  titulo: "JavaScript Avanzado",
  autor: "María López",
  año: 2024,
  paginas: 450
};

// Acceder
console.log(libro.titulo);

// Modificar
libro.año = 2025;

// Agregar
libro.editorial = "Tech Books";

// Eliminar
delete libro.paginas;

Operaciones básicas con objetos: crear, acceder, modificar y eliminar propiedades.

Objetos con Métodos

const calculadora = {
  valor: 0,
  
  sumar(n) {
    this.valor += n;
    return this; // Para encadenar
  },
  
  restar(n) {
    this.valor -= n;
    return this;
  },
  
  resultado() {
    return this.valor;
  }
};

// Encadenar métodos
const total = calculadora
  .sumar(10)
  .sumar(5)
  .restar(3)
  .resultado();
  
console.log(total); // 12

Objetos que contienen métodos y demostración de encadenamiento.

Desestructuración

const usuario = {
  nombre: "Laura",
  edad: 28,
  email: "laura@email.com",
  ciudad: "Barcelona"
};

// Desestructuración básica
const { nombre, edad } = usuario;
console.log(nombre); // "Laura"

// Con renombre
const { email: correo } = usuario;
console.log(correo); // "laura@email.com"

// Con valor por defecto
const { pais = "España" } = usuario;
console.log(pais); // "España"

Extraer propiedades de objetos de forma limpia y eficiente.

Objetos Anidados

const empresa = {
  nombre: "Tech Corp",
  fundacion: 2010,
  direccion: {
    calle: "Gran Vía 123",
    ciudad: "Madrid",
    pais: "España"
  },
  empleados: [
    { nombre: "Juan", puesto: "CEO" },
    { nombre: "María", puesto: "CTO" }
  ]
};

// Acceder a propiedades anidadas
console.log(empresa.direccion.ciudad);
// "Madrid"

console.log(empresa.empleados[0].nombre);
// "Juan"

Trabajar con estructuras de datos complejas usando objetos anidados.

Factory Functions (Funciones Fábrica)

// Función que crea objetos
function crearPersona(nombre, edad, profesion) {
  return {
    nombre,
    edad,
    profesion,
    presentarse() {
      return `Hola, soy ${this.nombre}, tengo ${this.edad} años y soy ${this.profesion}`;
    },
    cumplirAños() {
      this.edad++;
      console.log(`¡Feliz cumpleaños! Ahora tienes ${this.edad} años`);
    }
  };
}

// Crear múltiples objetos con la misma estructura
const persona1 = crearPersona("Carlos", 30, "Ingeniero");
const persona2 = crearPersona("Ana", 25, "Diseñadora");

console.log(persona1.presentarse());
persona2.cumplirAños();

Las funciones fábrica son un patrón para crear múltiples objetos con la misma estructura y comportamiento, útiles cuando necesitas crear muchas instancias similares.
